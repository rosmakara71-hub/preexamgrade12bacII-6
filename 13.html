👇 សូមពិនិត្យវិធីសាស្ត្រដោះស្រាយលម្អិតនៅខាងក្រោមសំនួរនីមួយៗ!

១. គណនាលីមីត៖ $\lim_{x \to 2} \frac{x^3 - 8}{x - 2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ប្រើរូបមន្តផលដកគូប $a^3 - b^3 = (a-b)(a^2+ab+b^2)$៖
$\lim_{x \to 2} \frac{(x-2)(x^2 + 2x + 4)}{x-2} = \lim_{x \to 2} (x^2 + 2x + 4)$
$= 2^2 + 2(2) + 4 = 4 + 4 + 4 = 12$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{\tan 3x}{\sin 5x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
បំប្លែងដោយប្រើលីមីតគំរូ $\lim_{x \to 0}\frac{\sin x}{x}=1$ និង $\lim_{x \to 0}\frac{\tan x}{x}=1$៖
$\lim_{x \to 0} \frac{\frac{\tan 3x}{3x} \times 3x}{\frac{\sin 5x}{5x} \times 5x} = \frac{3}{5} \times \frac{\lim_{x \to 0}\frac{\tan 3x}{3x}}{\lim_{x \to 0}\frac{\sin 5x}{5x}} = \frac{3}{5} \times \frac{1}{1} = \frac{3}{5}$។
៣. គណនាលីមីត៖ $\lim_{x \to 0} (1 + 3x)^{\frac{1}{x}}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $1^\infty$។ ប្រើរូបមន្តគ្រឹះ $\lim_{u \to 0} (1+u)^{\frac{1}{u}} = e$៖
តាង $u = 3x \implies \frac{1}{x} = \frac{3}{u}$។ នៅពេល $x \to 0 \implies u \to 0$។
$\lim_{u \to 0} (1 + u)^{\frac{3}{u}} = \left[\lim_{u \to 0} (1+u)^{\frac{1}{u}}\right]^3 = e^3$។
៤. រកតម្លៃ $a$ ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} \frac{\sqrt{x+1}-1}{x} & \text{បើ } x \neq 0 \\ a & \text{បើ } x = 0 \end{cases}$ ជាប់ត្រង់ $x = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ដើម្បីឱ្យអនុគមន៍ជាប់ត្រង់ $x=0$ លុះត្រាតែ $\lim_{x \to 0} f(x) = f(0) = a$។
គណនាលីមីតដោយគុណនឹងកន្សោមឆ្លាស់៖
$\lim_{x \to 0} \frac{(\sqrt{x+1}-1)(\sqrt{x+1}+1)}{x(\sqrt{x+1}+1)} = \lim_{x \to 0} \frac{x+1-1}{x(\sqrt{x+1}+1)}$
$= \lim_{x \to 0} \frac{x}{x(\sqrt{x+1}+1)} = \lim_{x \to 0} \frac{1}{\sqrt{x+1}+1} = \frac{1}{\sqrt{0+1}+1} = \frac{1}{2}$។
ដូចនេះ $a = \frac{1}{2}$។
៥. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \sin^3(2x)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេស្វ័យគុណបណ្តាក់ $(u^n)' = n \cdot u^{n-1} \cdot u'$៖
នៅទីនេះ $u = \sin(2x) \implies u' = (2x)'\cos(2x) = 2\cos(2x)$។
$f'(x) = 3 \cdot \sin^2(2x) \cdot [2\cos(2x)] = 6\sin^2(2x)\cos(2x)$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \frac{x}{\ln x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេផលចែក $\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$៖
$f'(x) = \frac{(x)'\ln x - x(\ln x)'}{(\ln x)^2} = \frac{1 \cdot \ln x - x \cdot \frac{1}{x}}{(\ln x)^2} = \frac{\ln x - 1}{(\ln x)^2}$។
៧. រកសមីការបន្ទាត់ប៉ះខ្សែគោង $y = e^x$ ត្រង់ចំណុចដែលមានអាប់ស៊ីស $x = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការបន្ទាត់ប៉ះ៖ $y = f'(0)(x - 0) + f(0)$
- គណនា $f(0) = e^0 = 1$។
- រកដេរីវេ $f'(x) = e^x \implies f'(0) = e^0 = 1$។
នាំឱ្យបាន៖ $y = 1(x - 0) + 1 \implies y = x + 1$។
៨. រកតម្លៃអតិបរមារបស់អនុគមន៍ $f(x) = x e^{-x}$ ចំពោះ $x \in \mathbb{R}$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រកដេរីវេ៖ $f'(x) = (x)'e^{-x} + x(e^{-x})' = e^{-x} - x e^{-x} = e^{-x}(1 - x)$។
ឱ្យ $f'(x) = 0 \implies 1 - x = 0 \implies x = 1$ (ព្រោះ $e^{-x} > 0$ ជានិច្ច)។
តម្លៃអតិបរមាគឺ $f(1) = 1 \cdot e^{-1} = \frac{1}{e}$។
៩. គណនាអាំងតេក្រាល៖ $\int \sin(3x) \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តគ្រឹះ $\int \sin(ax+b) \,dx = -\frac{1}{a}\cos(ax+b) + C$។
ដូចនេះ $\int \sin(3x) \,dx = -\frac{1}{3}\cos(3x) + C$។
១០. គណនាអាំងតេក្រាលកំណត់៖ $\int_{0}^{1} \frac{1}{1+x^2} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ព្រីមីទីវនៃ $\frac{1}{1+x^2}$ គឺ $\arctan x$។
$\int_{0}^{1} \frac{1}{1+x^2} \,dx = \left[\arctan x\right]_{0}^{1} = \arctan(1) - \arctan(0) = \frac{\pi}{4} - 0 = \frac{\pi}{4}$។
១១. គណនាអាំងតេក្រាលកំណត់៖ $\int_{1}^{2} x \ln x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើអាំងតេក្រាលដោយផ្នែក $\int u \,dv = uv - \int v \,du$៖
តាង $u = \ln x \implies du = \frac{1}{x}dx$
តាង $dv = x \,dx \implies v = \frac{x^2}{2}$
$\int_{1}^{2} x \ln x \,dx = \left[\frac{x^2}{2}\ln x\right]_{1}^{2} - \int_{1}^{2} \frac{x^2}{2} \cdot \frac{1}{x} \,dx$
$= \left(\frac{4}{2}\ln 2 - 0\right) - \frac{1}{2}\int_{1}^{2} x \,dx = 2\ln 2 - \frac{1}{2}\left[\frac{x^2}{2}\right]_{1}^{2}$
$= 2\ln 2 - \frac{1}{2}\left(2 - \frac{1}{2}\right) = 2\ln 2 - \frac{3}{4} = \ln(2^2) - \frac{3}{4} = \ln 4 - \frac{3}{4}$។
១២. គណនាផ្ទៃក្រឡាខណ្ឌដោយខ្សែគោង $y = e^x$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x = 0$ ដល់ $x = 1$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ផ្ទៃក្រឡា៖ $S = \int_{0}^{1} e^x \,dx = \left[e^x\right]_{0}^{1} = e^1 - e^0 = e - 1$ ឯកតាផ្ទៃ។
១៣. គណនាមាឌសូលីដបដិវត្តន៍បង្កើតឡើងដោយបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយបន្ទាត់ $y = x$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x=0$ ដល់ $x=2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តមាឌ៖ $V = \pi \int_{0}^{2} [f(x)]^2 \,dx = \pi \int_{0}^{2} x^2 \,dx$។
$= \pi \left[\frac{x^3}{3}\right]_{0}^{2} = \pi \left(\frac{2^3}{3} - 0\right) = \frac{8\pi}{3}$ ឯកតាមាឌ។
១៤. រកចម្លើយពិសេសនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' + 3y = 0$ បើដឹងថា $y(0) = 2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $y' + 3y = 0 \implies y' = -3y$ មានចម្លើយទូទៅ៖ $y = Ke^{-3x}$។
ប្រើលក្ខខណ្ឌដើម $y(0) = 2 \implies 2 = Ke^{-3(0)} \implies K = 2$។
ដូចនេះ ចម្លើយពិសេសគឺ $y = 2e^{-3x}$។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' - 4y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 - 4 = 0 \implies r^2 = 4 \implies r = \pm 2$ (ឫសពីរផ្សេងគ្នា)។
ដូចនេះ ចម្លើយទូទៅគឺ៖ $y = Ae^{2x} + Be^{-2x}$។
១៦. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' + 4y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 + 4 = 0 \implies r^2 = -4 \implies r = \pm 2i$ (ឫសជាកុំផ្លិចឆ្លាស់ $\alpha = 0, \beta = 2$)។
ដូចនេះ ចម្លើយទូទៅគឺ៖ $y = e^{0x}(A\cos 2x + B\sin 2x) = A\cos 2x + B\sin 2x$។
១៧. គេឱ្យចំនួនកុំផ្លិច $z = \frac{1+i}{1-i}$។ សរសេរ $z$ ជាទម្រង់ពីជគណិត។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គុណភាគយក និងភាគបែងនឹងកន្សោមឆ្លាស់របស់ភាគបែង $(1+i)$៖
$z = \frac{(1+i)(1+i)}{(1-i)(1+i)} = \frac{1 + 2i + i^2}{1^2 - i^2}$
ដោយសារ $i^2 = -1$ នាំឱ្យបាន៖ $z = \frac{1 + 2i - 1}{1 - (-1)} = \frac{2i}{2} = i$។
១៨. រកម៉ូឌុលនៃចំនួនកុំផ្លិច $z = 3 - 4i$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖

$|z| = \sqrt{3^2 + (-4)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$។
១៩. ក្នុងប្លង់កុំផ្លិច សំណុំចំណុច $M(z)$ ដែលផ្ទៀងផ្ទាត់លក្ខខណ្ឌ $|z - 2| = 3$ គឺជាអ្វី? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $|z - z_0| = R$ តំណាងឱ្យរង្វង់ដែលមានផ្ចិតចំណុចរូបភាព $z_0$ និងកាំ $R$។

ដូចនេះ វាជារង្វង់ផ្ចិត $(2,0)$ និងកាំ $3$។

💡 ដំណោះស្រាយលម្អិត៖
ដោយសារកំពូលនៅ $(0,0)$ ហើយកំនុំស្ថិតនៅលើយក្សអរដោនេ $Oy$ ត្រង់ $(0,2)$ នាំឱ្យប៉ារ៉ាបូលមានអ័ក្សឆ្លុះឈរ មានទម្រង់សមីការ $x^2 = 4py$។
យើងបានចម្ងាយពីកំពូលទៅកំនុំ $p = 2$។
ជំនួសតម្លៃ៖ $x^2 = 4(2)y \implies x^2 = 8y$។
២៣. រកកូអរដោនែកំនុំទាំងពីរនៃអេលីបដែលមានសមីការ៖ $\frac{x^2}{16} + \frac{y^2}{25} = 1$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
អេលីបមានអ័ក្សធំឈរ ព្រោះភាគបែងរបស់ $y^2$ ធំជាង នាំឱ្យ $a^2 = 25 \implies a = 5$ និង $b^2 = 16 \implies b = 4$។
គណនាចម្ងាយ $c = \sqrt{a^2 - b^2} = \sqrt{25 - 16} = \sqrt{9} = 3$។
ដោយសារវាជាអេលីបអ័ក្សធំឈរ ផ្ចិតនៅ $(0,0)$ នាំឱ្យកំនុំទាំងពីរមានកូអរដោនេ៖ $F(0, c)$ និង $F'(0, -c) \implies (0, 3)$ និង $(0, -3)$។
២៤. ក្នុងលំហកូអរដោនេ គេមានវ៉ិចទ័រ $\vec{u} = (1, 0, 2)$ និង $\vec{v} = (0, 1, 3)$។ គណនាផលិតផលវ៉ិចទ័រ $\vec{u} \times \vec{v}$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តដេទែរមីណង់នៃផលិតផលវ៉ិចទ័រ៖
$\vec{u} \times \vec{v} = \left( \begin{vmatrix} 0 & 2 \\ 1 & 3 \end{vmatrix}, -\begin{vmatrix} 1 & 2 \\ 0 & 3 \end{vmatrix}, \begin{vmatrix} 1 & 0 \\ 0 & 1 \end{vmatrix} \right)$
$= (0\cdot3 - 2\cdot1, -(1\cdot3 - 2\cdot0), 1\cdot1 - 0\cdot0) = (-2, -3, 1)$។
២៥. ក្នុងលំហកូអរដោនេ គណនាចម្ងាយពីចំណុច $A(1, 1, 1)$ ទៅប្លង់ $(P): x + 2y + 2z - 2 = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តចម្ងាយពីចំណុចទៅប្លង់៖ $d(A, P) = \frac{|ax_A + by_A + cz_A + d|}{\sqrt{a^2 + b^2 + c^2}}$
$d(A, P) = \frac{|1(1) + 2(1) + 2(1) - 2|}{\sqrt{1^2 + 2^2 + 2^2}} = \frac{|1 + 2 + 2 - 2|}{\sqrt{1 + 4 + 4}} = \frac{3}{\sqrt{9}} = \frac{3}{3} = 1$។
