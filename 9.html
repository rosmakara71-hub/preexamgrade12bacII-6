ពិន្ទុទទួលបាន៖ 0 / ១២៥

👇 សូមពិនិត្យវិធីសាស្ត្រដោះស្រាយលម្អិតនៅខាងក្រោមសំនួរនីមួយៗ!

១. គណនាលីមីត៖ $\lim_{x \to 2} \frac{x^3 - 8}{x^2 - 4}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ប្រើរូបមន្តបំបែកផលគុណកត្តា៖ $x^3 - 8 = (x-2)(x^2+2x+4)$ និង $x^2 - 4 = (x-2)(x+2)$។
សរសេរ៖ $\lim_{x \to 2} \frac{(x-2)(x^2+2x+4)}{(x-2)(x+2)} = \lim_{x \to 2} \frac{x^2+2x+4}{x+2} = \frac{2^2+2(2)+4}{2+2} = \frac{12}{4} = 3$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{\tan 3x}{\sin 5x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ប្រើប្រាស់លីមីតគំរូ៖ $\lim_{x \to 0} \frac{\tan ax}{ax} = 1$ និង $\lim_{x \to 0} \frac{\sin bx}{bx} = 1$។
បំបែកបាន៖ $\lim_{x \to 0} \frac{\frac{\tan 3x}{3x} \times 3x}{\frac{\sin 5x}{5x} \times 5x} = \frac{1 \times 3x}{1 \times 5x} = \frac{3}{5}$។
៣. គណនាលីមីត៖ $\lim_{x \to 0} \frac{e^{-2x} - 1}{x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ប្រើរូបមន្តលីមីតគ្រឹះ៖ $\lim_{u \to 0} \frac{e^u - 1}{u} = 1$។
សរសេរ៖ $\lim_{x \to 0} \frac{e^{-2x} - 1}{-2x} \times (-2) = 1 \times (-2) = -2$។
៤. រកតម្លៃ $k$ ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} \frac{\sqrt{x+4}-2}{x} & \text{បើ } x \neq 0 \\ k & \text{បើ } x = 0 \end{cases}$ ជាប់ត្រង់ចំណុច $x = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
អនុគមន៍ជាប់ត្រង់ $x = 0$ លុះត្រាតែ $\lim_{x \to 0} f(x) = f(0) = k$។
គណនាលីមីតដោយគុណកន្សោមឆ្លាស់ $\sqrt{x+4}+2$៖
$\lim_{x \to 0} \frac{(\sqrt{x+4}-2)(\sqrt{x+4}+2)}{x(\sqrt{x+4}+2)} = \lim_{x \to 0} \frac{x+4-4}{x(\sqrt{x+4}+2)} = \lim_{x \to 0} \frac{1}{\sqrt{x+4}+2} = \frac{1}{2+2} = \frac{1}{4}$។
ដូចនេះ $k = \frac{1}{4}$។
៥. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \frac{x}{e^x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេផលចែក $\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$ ដែល $u = x \implies u' = 1$ និង $v = e^x \implies v' = e^x$។
$f'(x) = \frac{1 \cdot e^x - x \cdot e^x}{(e^x)^2} = \frac{e^x(1-x)}{e^{2x}} = \frac{1-x}{e^x}$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \ln(\cos x)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេអនុគមន៍បណ្តាក់ $(\ln u)' = \frac{u'}{u}$ ដែល $u = \cos x \implies u' = -\sin x$។
$f'(x) = \frac{-\sin x}{\cos x} = -\tan x$។
៧. រកសមីការបន្ទាត់ប៉ះខ្សែគោង $y = x^3 - 2x$ ត្រង់ចំណុចដែលមានអាប់ស៊ីស $x = 1$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការបន្ទាត់ប៉ះ៖ $y = f'(1)(x - 1) + f(1)$
- តម្លៃអនុគមន៍៖ $f(1) = 1^3 - 2(1) = -1$។
- មេគុណប្រាប់ទិស៖ $f'(x) = 3x^2 - 2 \implies f'(1) = 3(1)^2 - 2 = 1$។
នាំឱ្យបានសមីការ៖ $y = 1(x - 1) - 1 \implies y = x - 2$។
៨. ស្វែងរកចំណុចបរមារបស់អនុគមន៍៖ $f(x) = \frac{1}{3}x^3 - 2x^2 + 3x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គណនាដេរីវេ៖ $f'(x) = x^2 - 4x + 3$។
ឱ្យ $f'(x) = 0 \implies (x-1)(x-3) = 0 \implies x = 1$ ឬ $x = 3$។
តាមតារាងសញ្ញា $f'(x)$៖
- $f'(x)$ ប្តូរសញ្ញាពី $(+)$ ទៅ $(-)$ ត្រង់ $x = 1 \implies$ អតិបរមា។
- $f'(x)$ ប្តូរសញ្ញាពី $(-)$ ទៅ $(+)$ ត្រង់ $x = 3 \implies$ អប្បបរមា។
៩. គណនាអាំងតេក្រាលមិនកំណត់៖ $\int (e^x + \sin x) \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមលក្ខណៈគ្រឹះនៃព្រីមីទីវ៖
- $\int e^x dx = e^x + C$
- $\int \sin x dx = -\cos x + C$
ដូចនេះ លទ្ធផលគឺ $e^x - \cos x + C$។
១០. គណនាអាំងតេក្រាលកំណត់ដោយវិធីប្តូរអថេរ៖ $\int_{0}^{1} x e^{x^2} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = x^2 \implies du = 2x dx \implies x dx = \frac{1}{2} du$។
ប្តូរគោល៖ បើ $x = 0 \implies u = 0$ ៖ បើ $x = 1 \implies u = 1$។
នាំឱ្យបាន៖ $\int_{0}^{1} e^u \cdot \frac{1}{2} du = \frac{1}{2} [e^u]_{0}^{1} = \frac{1}{2}(e^1 - e^0) = \frac{1}{2}(e - 1)$។
១១. គណនាអាំងតេក្រាលកំណត់ដោយផ្នែក៖ $\int_{1}^{e} \ln x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = \ln x \implies du = \frac{1}{x} dx$ និង $dv = dx \implies v = x$។
តាមរូបមន្ត $\int u dv = uv - \int v du$ នាំឱ្យបានព្រីមីទីវ៖ $x\ln x - \int x \cdot \frac{1}{x} dx = x\ln x - x$។
គណនាតម្លៃ៖ $[x\ln x - x]_{1}^{e} = (e\ln e - e) - (1\ln 1 - 1) = (e - e) - (0 - 1) = 0 - (-1) = 1$។
១២. គណនាផ្ទៃក្រឡាខណ្ឌដោយខ្សែគោង $y = 4 - x^2$ និងអ័ក្សអាប់ស៊ីស $Ox$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រកចំណុចប្រសព្វនឹងអ័ក្ស $Ox$៖ $4 - x^2 = 0 \implies x = \pm 2$។
ផ្ទៃក្រឡា៖ $S = \int_{-2}^{2} (4 - x^2) dx = \left[ 4x - \frac{x^3}{3} \right]_{-2}^{2}$
$= \left(4(2) - \frac{2^3}{3}\right) - \left(4(-2) - \frac{(-2)^3}{3}\right) = \left(8 - \frac{8}{3}\right) - \left(-8 + \frac{8}{3}\right) = \frac{16}{3} - \left(-\frac{16}{3}\right) = \frac{32}{3}$ ឯកតាផ្ទៃ។
១៣. គណនាមាឌសូលីដបដិវត្តន៍បង្កើតឡើងដោយបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយខ្សែគោង $y = \frac{1}{x}$ បន្ទាត់ឈរ $x=1$ និង $x=2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តមាឌ៖ $V = \pi \int_{1}^{2} [f(x)]^2 dx = \pi \int_{1}^{2} \left(\frac{1}{x}\right)^2 dx = \pi \int_{1}^{2} \frac{1}{x^2} dx$។
$= \pi \left[ -\frac{1}{x} \right]_{1}^{2} = \pi \left( -\frac{1}{2} - (-1) \right) = \pi \left( 1 - \frac{1}{2} \right) = \frac{\pi}{2}$ ឯកតាមាឌ។
១៤. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' + 3y = 6$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការមានទម្រង់ $y' + ay = b$ ដែលមានចម្លើយទូទៅគឺ $y = Ke^{-ax} + \frac{b}{a}$។
ទីនេះ $a = 3, b = 6 \implies \frac{b}{a} = \frac{6}{3} = 2$។ ដូចនេះ $y = Ke^{-3x} + 2$។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' - 4y' + 5y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 - 4r + 5 = 0$ មាន $\Delta' = (-2)^2 - 1(5) = 4 - 5 = -1 = i^2$។
ឫសសមីការគឺកុំផ្លិចឆ្លាស់៖ $r = 2 \pm i$ (ផ្នែកពិត $\alpha = 2$, ផ្នែកនិម្មិត $\beta = 1$)។
ដូចនេះចម្លើយទូទៅគឺ៖ $y = e^{2x}(A\cos x + B\sin x)$។
១៦. រកចម្លើយពិសេសនៃសមីការឌីផេរ៉ង់ស្យែល $y' = 2x + 1$ បើដឹងថាក្រាបខ្សែគោងកាត់តាមចំណុច $A(1, 3)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រកចម្លើយទូទៅដោយរកអាំងតេក្រាល៖ $y = \int (2x + 1) dx = x^2 + x + C$។
ប្រើលក្ខខណ្ឌកាត់តាមចំណុច $A(1, 3) \implies 3 = 1^2 + 1 + C \implies 3 = 2 + C \implies C = 1$។
ដូចនេះចម្លើយពិសេសគឺ $y = x^2 + x + 1$។
១៧. សរសេរចំនួនកុំផ្លិច $z = \frac{2 + i}{2 - i}$ ជាទម្រង់ពិជគណិត $a + bi$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គុណនឹងកន្សោមឆ្លាស់នៃភាគបែងគឺ $(2+i)$៖
$z = \frac{(2+i)(2+i)}{(2-i)(2+i)} = \frac{4 + 4i + i^2}{2^2 - i^2} = \frac{4 + 4i - 1}{4 + 1} = \frac{3 + 4i}{5} = \frac{3}{5} + \frac{4}{5}i$។
១៨. គណនាម៉ូឌុលនៃចំនួនកុំផ្លិច៖ $z = (1 + i\sqrt{3})^4$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើលក្ខណៈនៃម៉ូឌុល៖ $|z^n| = |z|^n$។
- រកម៉ូឌុលខាងក្នុង៖ $|1 + i\sqrt{3}| = \sqrt{1^2 + (\sqrt{3})^2} = \sqrt{1+3} = 2$។
- នាំឱ្យ៖ $|z| = 2^4 = 16$។
១៩. បំលែងចំនួនកុំផ្លិច $z = 1 - i$ ទៅជាទម្រង់ត្រីកោណមាត្រ។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ម៉ូឌុល៖ $r = \sqrt{1^2 + (-1)^2} = \sqrt{2}$។
- ទម្រង់ត្រីកោណមាត្រ៖ $z = \sqrt{2}\left(\frac{1}{\sqrt{2}} - i\frac{1}{\sqrt{2}}\right) = \sqrt{2}\left(\cos\frac{\pi}{4} - i\sin\frac{\pi}{4}\right)$។
ដោយប្រើរូបមន្តមុំផ្ទុយ៖ ដូចនេះ $z = \sqrt{2}\left(\cos\left(-\frac{\pi}{4}\right) + i\sin\left(-\frac{\pi}{4}\right)\right)$។
២០. ក្នុងប្លង់កុំផ្លិច សំណុំចំណុច $M(z)$ ដែលផ្ទៀងផ្ទាត់លក្ខខណ្ឌ $\text{Re}(z) = 2$ គឺជាអ្វី? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $z = x + iy$ ដែល $x, y \in \mathbb{R}$។ ផ្នែកពិតគឺ $\text{Re}(z) = x$។
តាមលក្ខខណ្ឌ $\text{Re}(z) = 2 \implies x = 2$។
សមីការ $x = 2$ ជាសមីការបន្ទាត់ត្រង់ឈរស្របនឹងអ័ក្សអរដោនេ។
២១. តើគេអាចបង្កើតលេខដែលមាន ៣ ខ្ទង់ផ្សេងគ្នា បានចំនួនប៉ុន្មានរបៀប ចេញពីលេខចំណោម $\{1, 2, 3, 4, 5\}$? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
លំហាត់នេះប្រើ ច្រលាស់ (Permutation) ព្រោះលំដាប់នៃខ្ទង់លេខមានសារៈសំខាន់ និងមិនមានការដដែលៗ៖
$P(5, 3) = \frac{5!}{(5-3)!} = \frac{5 \times 4 \times 3 \times 2!}{2!} = 5 \times 4 \times 3 = 60$ របៀប។
២២. ក្នុងថង់មួយមានបាល់ក្រហម ៤ និងបាល់ខៀវ ៦។ គេចាប់យកបាល់ ២ ព្រមគ្នាដោយចៃដន្យ។ រកប្រូបាបដែលចាប់បានបាល់ខុសពណ៌គ្នា (ក្រហមមួយ ខៀវមួយ)។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ចំនួនករណីអាច៖ $n(S) = C(10, 2) = \frac{10 \times 9}{2} = 45$។
- ចំនួនករណីស្រប (ក្រហម១ និងខៀវ១)៖ $n(A) = C(4, 1) \times C(6, 1) = 4 \times 6 = 24$។
- ប្រូបាប៖ $P(A) = \frac{24}{45} = \frac{8}{15}$។
២៣. រកសមីការអាស៊ីមតូតទាំងពីរនៃអ៊ីពែបូលដែលមានសមីការ៖ $\frac{x^2}{16} - \frac{y^2}{9} = 1$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការស្តង់ដាអ៊ីពែបូលមានអ័ក្សទទឹងដេក៖ $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ មានអាស៊ីមតូត៖ $y = \pm \frac{b}{a}x$។
តាមសមីការលំហាត់៖ $a^2 = 16 \implies a = 4$ និង $b^2 = 9 \implies b = 3$។
ដូចនេះ សមីការអាស៊ីមតូតគឺ $y = \pm \frac{3}{4}x$។
២៤. ក្នុងលំហកូអរដោនេ រកសមីការប៉ារ៉ាម៉ែត្រនៃបន្ទាត់កាត់តាមចំណុច $A(1, 2, 3)$ និងមានវ៉ិចទ័រប្រាប់ទិស $\vec{u} = (-1, 0, 2)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រូបមន្តសមីការប៉ារ៉ាម៉ែត្រនៃបន្ទាត់៖ $\begin{cases} x = x_A + a t \\ y = y_A + b t \\ z = z_A + c t \end{cases}$
ជំនួស $A(1, 2, 3)$ និង $\vec{u}(-1, 0, 2)$ ចូល៖
$\begin{cases} x = 1 - t \\ y = 2 + 0t \\ z = 3 + 2t \end{cases} \implies \begin{cases} x = 1 - t \\ y = 2 \\ z = 3 + 2t \end{cases}$ ($t \in \mathbb{R}$)។
២៥. គណនាចម្ងាយពីចំណុច $A(1, 1, 1)$ ទៅប្លង់ $(P): 2x - 2y + z + 5 = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តចម្ងាយពីចំណុចទៅប្លង់៖ $d(A, P) = \frac{|a x_A + b y_A + c z_A + d|}{\sqrt{a^2 + b^2 + c^2}}$
$d(A, P) = \frac{|2(1) - 2(1) + 1(1) + 5|}{\sqrt{2^2 + (-2)^2 + 1^2}} = \frac{|2 - 2 + 1 + 5|}{\sqrt{4 + 4 + 1}} = \frac{6}{\sqrt{9}} = \frac{6}{3} = 2$។
