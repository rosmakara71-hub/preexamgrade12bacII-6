👇 សូមពិនិត្យវិធីសាស្ត្រដោះស្រាយលម្អិតនៅខាងក្រោមសំនួរនីមួយៗ!

១. គណនាលីមីត៖ $\lim_{x \to 2} \frac{x^3 - 8}{x - 2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ប្រើរូបមន្តផលដកគូប៖ $x^3 - 8 = (x - 2)(x^2 + 2x + 4)$។
$\lim_{x \to 2} \frac{(x-2)(x^2+2x+4)}{x-2} = \lim_{x \to 2} (x^2 + 2x + 4) = 2^2 + 2(2) + 4 = 12$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{\sqrt{1+3x} - 1}{x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គុណភាគយក និងភាគបែងនឹងកន្សោមឆ្លាស់ $\sqrt{1+3x} + 1$៖
$\lim_{x \to 0} \frac{(1+3x) - 1}{x(\sqrt{1+3x} + 1)} = \lim_{x \to 0} \frac{3x}{x(\sqrt{1+3x} + 1)} = \lim_{x \to 0} \frac{3}{\sqrt{1+3x} + 1} = \frac{3}{1+1} = \frac{3}{2}$។
៣. គណនាលីមីត៖ $\lim_{x \to 0} \frac{\tan 3x}{\sin 5x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ចែកភាគយក និងភាគបែងនឹង $x$ រួចប្រើលីមីតគំរូ $\lim_{u \to 0} \frac{\sin u}{u} = 1$ និង $\lim_{u \to 0} \frac{\tan u}{u} = 1$៖
$\lim_{x \to 0} \frac{\frac{\tan 3x}{3x} \times 3}{\frac{\sin 5x}{5x} \times 5} = \frac{1 \times 3}{1 \times 5} = \frac{3}{5}$។
៤. គណនាលីមីត៖ $\lim_{x \to +\infty} \frac{2x^2 - x + 1}{3x^2 + 5}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ទាញតួដែលមានដឺក្រេខ្ពស់ជាងគេជាកត្តារួម (ឬរកផលធៀបមេគុណដឺក្រេខ្ពស់បំផុត)៖
$\lim_{x \to +\infty} \frac{2x^2}{3x^2} = \frac{2}{3}$។
៥. រកដេរីវេនៃអនុគមន៍៖ $f(x) = x \ln x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តផលគុណ $(uv)' = u'v + uv'$៖
$f'(x) = (x)' \ln x + x (\ln x)' = 1 \cdot \ln x + x \cdot \frac{1}{x} = \ln x + 1$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \frac{e^x}{x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តផលចែក $\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$៖
$f'(x) = \frac{(e^x)'x - e^x(x)'}{x^2} = \frac{e^x \cdot x - e^x \cdot 1}{x^2} = \frac{e^x(x-1)}{x^2}$។
៧. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \sin^3 x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តស្វ័យគុណបណ្តាក់ $(u^n)' = n \cdot u^{n-1} \cdot u'$៖
$f'(x) = 3\sin^2 x \cdot (\sin x)' = 3\sin^2 x \cos x$។
៨. រកកូអរដោនេចំណុចរបត់ (Inflection Point) នៃខ្សែគោងតំណាងអនុគមន៍ $f(x) = x^3 - 3x^2 + 2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- រកដេរីវេទីមួយ៖ $f'(x) = 3x^2 - 6x$
- រកដេរីវេទីពីរ៖ $f''(x) = 6x - 6$
- ឲ្យ $f''(x) = 0 \implies 6x - 6 = 0 \implies x = 1$។
- គណនាអរដោនេ៖ $f(1) = 1^3 - 3(1)^2 + 2 = 1 - 3 + 2 = 0$។
ដូចនេះ ចំណុចរបត់គឺ $(1, 0)$។
៩. គណនាអាំងតេក្រាលមិនកំណត់៖ $\int (3x^2 - 4x + 5) \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើព្រីមីទីវគ្រឹះ $\int x^n \,dx = \frac{x^{n+1}}{n+1} + C$៖
$\int (3x^2 - 4x + 5) \,dx = 3\left(\frac{x^3}{3}\right) - 4\left(\frac{x^2}{2}\right) + 5x + C = x^3 - 2x^2 + 5x + C$។
១០. គណនាអាំងតេក្រាលកំណត់៖ $\int_{0}^{1} e^{2x} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ព្រីមីទីវនៃ $e^{2x}$ គឺ $\frac{1}{2}e^{2x}$។
$\int_{0}^{1} e^{2x} \,dx = \left[\frac{1}{2}e^{2x}\right]_{0}^{1} = \frac{1}{2}e^{2(1)} - \frac{1}{2}e^{2(0)} = \frac{e^2 - 1}{2}$។
១១. គណនាអាំងតេក្រាលកំណត់៖ $\int_{0}^{\frac{\pi}{2}} \sin x \cos x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើការប្ដូរអថេរ៖ តាង $u = \sin x \implies du = \cos x \,dx$។
- បើ $x = 0 \implies u = \sin 0 = 0$
- បើ $x = \frac{\pi}{2} \implies u = \sin\frac{\pi}{2} = 1$
$\int_{0}^{1} u \,du = \left[\frac{u^2}{2}\right]_{0}^{1} = \frac{1^2}{2} - 0 = \frac{1}{2}$។
១២. គណនាផ្ទៃក្រឡាខណ្ឌដោយខ្សែគោង $y = \frac{1}{x}$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x = 1$ ដល់ $x = e$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ផ្ទៃក្រឡា៖ $S = \int_{1}^{e} \frac{1}{x} \,dx = \left[\ln|x|\right_{1}^{e} = \ln e - \ln 1 = 1 - 0 = 1$ ឯកតាផ្ទៃ។
១៣. គណនាមាឌសូលីដបដិវត្តន៍បង្កើតឡើងដោយការបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយបន្ទាត់ $y = x$ និងបន្ទាត់ឈរ $x = 0$ ដល់ $x = 2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តមាឌ៖ $V = \pi \int_{a}^{b} [f(x)]^2 \,dx$។
$V = \pi \int_{0}^{2} x^2 \,dx = \pi \left[\frac{x^3}{3}\right]_{0}^{2} = \pi \left(\frac{8}{3} - 0\right) = \frac{8\pi}{3}$ ឯកតាមាឌ។
១៤. រកចម្លើយពិសេសនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' + 3y = 0$ បើដឹងថា $y(0) = 2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $y' + 3y = 0 \implies y' = -3y$ មានចម្លើយទូទៅ៖ $y = Ke^{-3x}$។
តាមលក្ខខណ្ឌដើម $y(0) = 2 \implies 2 = Ke^{-3(0)} \implies K = 2$។
ដូចនេះ ចម្លើយពិសេសគឺ $y = 2e^{-3x}$។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' - 4y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈគឺ៖ $r^2 - 4 = 0 \implies r = \pm 2$។
ឫសជាចំនួនពិតពីរផ្សេងគ្នា $r_1=2, r_2=-2$ នាំឱ្យចម្លើយទូទៅគឺ៖ $y = Ae^{2x} + Be^{-2x}$ ($A, B \in \mathbb{R}$)។
១៦. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' + 4y' + 4y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 + 4r + 4 = 0 \implies (r+2)^2 = 0 \implies r_0 = -2$ (ឫសឌុប)។
ចម្លើយទូទៅមានទម្រង់៖ $y = (Ax+B)e^{r_0x} = (Ax+B)e^{-2x}$។
១៧. សរសេរចំនួនកុំផ្លិច៖ $z = \frac{1+i}{1-i}$ ជាទម្រង់ពីជគណិត $a+bi$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គុណភាគយក និងភាគបែងនឹងកន្សោមឆ្លាស់នៃភាគបែងគឺ $(1+i)$៖
$z = \frac{(1+i)(1+i)}{(1-i)(1+i)} = \frac{1 + 2i + i^2}{1^2 - i^2} = \frac{1 + 2i - 1}{1 + 1} = \frac{2i}{2} = i$។
១៨. រកអាគុយម៉ង់ចម្បង (Argument) នៃចំនួនកុំផ្លិច $z = -1 + i$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ម៉ូឌុល $r = \sqrt{(-1)^2 + 1^2} = \sqrt{2}$។
- គណនា $\cos\theta = \frac{-1}{\sqrt{2}} = -\frac{\sqrt{2}}{2}$ និង $\sin\theta = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$។
- ចំណុចស្ថិតនៅក្នុងកាដ្រង់ទី២ នាំឱ្យ $\theta = \pi - \frac{\pi}{4} = \frac{3\pi}{4}$។
១៩. គណនាម៉ូឌុលនៃចំនួនកុំផ្លិច $z = (3 + 4i)^2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមលក្ខណៈម៉ូឌុល៖ $|z^n| = |z|^n$។
$|3 + 4i| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$។
ដូចនេះ $|z| = 5^2 = 25$។
២០. តើមានគន្លងខុសៗគ្នាគិតតាមលំដាប់ចំនួនប៉ុន្មានក្នុងការជ្រើសរើសអក្សរ ៤ ចេញពីសំណុំអក្សរ $\{A, B, C, D, E, F\}$? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ការជ្រើសរើសដោយគិតគូរពីលំដាប់ គឺជាចរន្ត (Permutation) នៃធាតុ ៤ យកចេញពីធាតុ ៦៖
$P(6,4) = \frac{6!}{(6-4)!} = \frac{6!}{2!} = 6 \times 5 \times 4 \times 3 = 360$ របៀប។
២១. គេបោះកាក់ធម្មតាមួយចំនួន ៣ ដង។ រកប្រូបាបដើម្បីឱ្យកាក់ចេញមុខខាងក្បាល (H) ចំនួន ២ ដងគត់។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- លំហសំណាក $S = \{HHH, HHT, HTH, THH, HTT, THT, TTH, TTT\} \implies n(S) = 8$។
- ព្រឹត្តិការណ៍ $A$ ចេញក្បាល ២ ដង៖ $A = \{HHT, HTH, THH\} \implies n(A) = 3$។
- ប្រូបាប៖ $P(A) = \frac{n(A)}{n(S)} = \frac{3}{8}$។
២២. រកកូអរដោនេកំនុំ (Focus) នៃប៉ារ៉ាបូលដែលមានសមីការស្តង់ដា $y^2 = 8x$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការមានទម្រង់ $y^2 = 4px$ ដែលមានកំពូលនៅគល់ $O(0,0)$ និងអ័ក្សឆ្លុះដេក។
យើងបាន $4p = 8 \implies p = 2$។
ដូចនេះ កំនុំស្ថិតនៅចំណុច $F(p, 0) = F(2, 0)$។
២៣. រកកំពូលទាំងពីរនៃអ៊ីពែបូលដែលមានសមីការ៖ $\frac{x^2}{9} - \frac{y^2}{16} = 1$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
អ៊ីពែបូលមានអ័ក្សទទឹងដេក (អ័ក្ស $Ox$) ព្រោះតួ $x^2$ វិជ្ជមាន។
យើងមាន $a^2 = 9 \implies a = 3$។
ដូចនេះ កំពូលទាំងពីរគឺ $V(\pm a, 0) = V(\pm 3, 0)$។
២៤. ក្នុងលំហកូអរដោនេ គណនាចម្ងាយរវាងចំណុច $A(1, 2, 3)$ និង $B(4, 2, -1)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តចម្ងាយរវាងពីរចំណុច៖ $d = \sqrt{(x_B - x_A)^2 + (y_B - y_A)^2 + (z_B - z_A)^2}$។
$AB = \sqrt{(4-1)^2 + (2-2)^2 + (-1-3)^2} = \sqrt{3^2 + 0^2 + (-4)^2} = \sqrt{9 + 16} = \sqrt{25} = 5$។
