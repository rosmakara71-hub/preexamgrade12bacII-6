ពិន្ទុទទួលបាន៖ 0 / ១២៥

👇 សូមពិនិត្យវិធីសាស្ត្រដោះស្រាយលម្អិតនៅខាងក្រោមសំនួរនីមួយៗ!

១. គណនាលីមីត៖ $\lim_{x \to 1} \frac{\sqrt{x+3} - 2}{x - 1}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ គុណភាគយកនិងភាគបែងនឹងកន្សោមឆ្លាស់ $\sqrt{x+3} + 2$៖
$\lim_{x \to 1} \frac{(\sqrt{x+3}-2)(\sqrt{x+3}+2)}{(x-1)(\sqrt{x+3}+2)} = \lim_{x \to 1} \frac{x+3-4}{(x-1)(\sqrt{x+3}+2)} = \lim_{x \to 1} \frac{x-1}{(x-1)(\sqrt{x+3}+2)}$
$= \lim_{x \to 1} \frac{1}{\sqrt{x+3}+2} = \frac{1}{\sqrt{1+3}+2} = \frac{1}{2+2} = \frac{1}{4}$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{1 - \cos 4x}{x^2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ប្រើរូបមន្តត្រីកោណមាត្រ៖ $1 - \cos 4x = 2\sin^2 2x$។
សរសេរ៖ $\lim_{x \to 0} \frac{2\sin^2 2x}{x^2} = 2 \cdot \lim_{x \to 0} \left(\frac{\sin 2x}{2x} \cdot 2\right)^2 = 2 \cdot (1 \cdot 2)^2 = 2 \cdot 4 = 8$។
៣. គណនាលីមីត៖ $\lim_{x \to +\infty} \frac{2x - \sin x}{x + \cos x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ចាប់ $x$ ជាកត្តារួមទាំងភាគយកនិងភាគបែង៖
$\lim_{x \to +\infty} \frac{x(2 - \frac{\sin x}{x})}{x(1 + \frac{\cos x}{x})} = \lim_{x \to +\infty} \frac{2 - \frac{\sin x}{x}}{1 + \frac{\cos x}{x}}$។
ដោយសារ $\lim_{x \to +\infty} \frac{\sin x}{x} = 0$ និង $\lim_{x \to +\infty} \frac{\cos x}{x} = 0$ (អនុគមន៍មានដែនកំណត់ចែកនឹងអនន្ត)។
នាំឱ្យលីមីតស្មើនឹង $\frac{2 - 0}{1 + 0} = 2$។
៤. គណនាលីមីត៖ $\lim_{x \to 0} (1 + 3x)^{\frac{1}{x}}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $1^\infty$។ ប្រើរូបមន្តគ្រឹះ៖ $\lim_{u \to 0} (1+u)^{\frac{1}{u}} = e$។
សរសេរលីមីតជា៖ $\lim_{x \to 0} \left[(1 + 3x)^{\frac{1}{3x}}\right]^3 = e^3$។
៥. រកតម្លៃ $a$ ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} \frac{x^2 - 9}{x - 3} & \text{បើ } x \neq 3 \\ a & \text{បើ } x = 3 \end{cases}$ ជាប់ត្រង់ចំណុច $x = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ដើម្បីឱ្យអនុគមន៍ $f$ ជាប់ត្រង់ $x = 3$ លុះត្រាតែ $\lim_{x \to 3} f(x) = f(3)$។
- យើងមាន $f(3) = a$
- គណនាលីមីត៖ $\lim_{x \to 3} \frac{x^2 - 9}{x - 3} = \lim_{x \to 3} \frac{(x-3)(x+3)}{x-3} = \lim_{x \to 3} (x+3) = 3 + 3 = 6$។
ដូចនេះ $a = 6$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = x \ln x - x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេផលគុណ $(uv)' = u'v + uv'$ ចំពោះតួទីមួយ៖
$f'(x) = (x)'\ln x + x(\ln x)' - (x)' = 1 \cdot \ln x + x \cdot \frac{1}{x} - 1$
$= \ln x + 1 - 1 = \ln x$។
៧. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \cos^3(2x)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តស្វ័យគុណនៃអនុគមន៍បណ្តាក់ $(u^n)' = n \cdot u^{n-1} \cdot u'$៖
$f'(x) = 3\cos^2(2x) \cdot (\cos 2x)' = 3\cos^2(2x) \cdot (-2\sin 2x) = -6\cos^2(2x)\sin(2x)$។
៨. រកសមីការអាស៊ីមតូតទ្រេតនៃអនុគមន៍៖ $f(x) = \frac{x^2 + 2x - 1}{x + 1}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
បំបែកប្រភាគដោយវិធីចែកពហុធា ឬសរសេរ៖
$f(x) = \frac{x^2 + 2x + 1 - 2}{x + 1} = \frac{(x+1)^2 - 2}{x + 1} = x + 1 - \frac{2}{x + 1}$។
ដោយសារ $\lim_{x \to \pm\infty} \frac{2}{x+1} = 0$ ដូចនេះបន្ទាត់ $y = x + 1$ ជាអាស៊ីមតូតទ្រេត។
៩. រកសមីការបន្ទាត់ប៉ះខ្សែគោង $y = x^2 - 3x + 2$ ត្រង់ចំណុចដែលមានអាប់ស៊ីស $x = 2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការបន្ទាត់ប៉ះមានទម្រង់៖ $y = f'(x_0)(x - x_0) + f(x_0)$ ដែល $x_0 = 2$។
- រកតង់សង់៖ $f(2) = 2^2 - 3(2) + 2 = 4 - 6 + 2 = 0$។
- រកមេគុណប្រាប់ទិស៖ $f'(x) = 2x - 3 \implies f'(2) = 2(2) - 3 = 1$។
នាំឱ្យបានសមីការបន្ទាត់ប៉ះ៖ $y = 1(x - 2) + 0 \implies y = x - 2$។
១០. គណនាអាំងតេក្រាលមិនកំណត់៖ $\int (3x^2 - 4x + 5) \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តគ្រឹះ $\int x^n dx = \frac{x^{n+1}}{n+1} + C$៖
$\int (3x^2 - 4x + 5) dx = 3\left(\frac{x^3}{3}\right) - 4\left(\frac{x^2}{2}\right) + 5x + C = x^3 - 2x^2 + 5x + C$។
១១. គណនាអាំងតេក្រាលកំណត់៖ $\int_{0}^{\ln 2} e^{2x} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ព្រីមីទីវនៃ $e^{2x}$ គឺ $\frac{1}{2}e^{2x}$។
$\int_{0}^{\ln 2} e^{2x} dx = \left[ \frac{1}{2}e^{2x} \right]_{0}^{\ln 2} = \frac{1}{2}\left(e^{2\ln 2} - e^0\right) = \frac{1}{2}\left(e^{\ln 4} - 1\right) = \frac{1}{2}(4 - 1) = \frac{3}{2}$។
១២. គណនាអាំងតេក្រាលដោយប្រើវិធីសាស្ត្រផ្នែក៖ $\int x e^x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = x \implies du = dx$ និង $dv = e^x dx \implies v = e^x$។
តាមរូបមន្ត $\int u \,dv = uv - \int v \,du$៖
$\int x e^x dx = x e^x - \int e^x dx = x e^x - e^x + C = (x - 1)e^x + C$។
១៣. គណនាមាឌសូលីដបដិវត្តន៍ បង្កើតឡើងដោយការបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយខ្សែគោង $y = \sqrt{x}$ និងបន្ទាត់ឈរ $x = 0$, $x = 4$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តមាឌ៖ $V = \pi \int_{a}^{b} [f(x)]^2 dx$។
$V = \pi \int_{0}^{4} (\sqrt{x})^2 dx = \pi \int_{0}^{4} x \,dx = \pi \left[ \frac{x^2}{2} \right]_{0}^{4} = \pi \left(\frac{16}{2} - 0\right) = 8\pi$ ឯកតាមាឌ។
១៤. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' - 3y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការឌីផេរ៉ង់ស្យែលលីនេអ៊ែរលំដាប់ទី១ ទម្រង់ទូទៅ $y' - ay = 0$ មានចម្លើយទូទៅគឺ $y = Ke^{ax}$។
នៅទីនេះ $a = 3$ ដូចនេះចម្លើយគឺ $y = Ke^{3x}$ ($K$ ជាចំនួនពិតថេរ)។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' + 4y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការទម្រង់ $y'' + \omega^2 y = 0$ ដែលមានសមីការលក្ខណៈ $r^2 + \omega^2 = 0 \implies r = \pm \omega i$។
ទីនេះ $\omega^2 = 4 \implies \omega = 2$ (ឫសជាចំនួនកុំផ្លិចឆ្លាស់គ្មានផ្នែកពិត)។
ដូចនេះចម្លើយទូទៅគឺ៖ $y = A\cos 2x + B\sin 2x$ ($A, B$ ជាចំនួនថេរ)។
១៦. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' - 2y' + y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 - 2r + 1 = 0 \implies (r - 1)^2 = 0 \implies r_0 = 1$ (ឫសឌុប)។
ករណីសមីការលក្ខណៈមានឫសឌុប ចម្លើយទូទៅគឺមានទម្រង់៖ $y = (C_1x + C_2)e^{r_0x} = (C_1x + C_2)e^x$។
១៧. រកទម្រង់ពិជគណិតនៃចំនួនកុំផ្លិច៖ $z = (1 + 2i)(3 - i)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គុណពន្លាតធម្មតាដោយប្រើលក្ខណៈ $i^2 = -1$៖
$z = 1(3) - 1(i) + 2i(3) - 2i(i) = 3 - i + 6i - 2i^2$
$= 3 + 5i - 2(-1) = 3 + 5i + 2 = 5 + 5i$។
១៨. រកឫសការេនៃចំនួនកុំផ្លិច៖ $z = -3 + 4i$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $w = x + iy$ ជាឫសការេនៃ $z \implies (x + iy)^2 = -3 + 4i$
យើងបានប្រព័ន្ធសមីការ៖ $\begin{cases} x^2 - y^2 = -3 \\ 2xy = 4 \\ x^2 + y^2 = |-3+4i| = \sqrt{(-3)^2+4^2} = 5 \end{cases}$
បូកសមីការទី១ និងទី៣៖ $2x^2 = 2 \implies x^2 = 1 \implies x = \pm 1$។
ដោយសារ $2xy = 4 > 0 \implies x$ និង $y$ មានសញ្ញាដូចគ្នា។
- បើ $x = 1 \implies y = 2 \implies w = 1 + 2i$។
- បើ $x = -1 \implies y = -2 \implies w = -1 - 2i$។ ដូចនេះ $w = \pm(1 + 2i)$។
១៩. រកអាគុយម៉ង់ចម្បងនៃចំនួនកុំផ្លិច៖ $z = -1 - i\sqrt{3}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ម៉ូឌុល៖ $|z| = \sqrt{(-1)^2 + (-\sqrt{3})^2} = \sqrt{1 + 3} = 2$។
- ទម្រង់ត្រីកោណមាត្រ៖ $z = 2\left(-\frac{1}{2} - i\frac{\sqrt{3}}{2}\right)$។
ដោយសារ $\cos\theta = -\frac{1}{2}$ និង $\sin\theta = -\frac{\sqrt{3}}{2}$ (ស្ថិតនៅក្នុងកាដ្រង់ទី៣)។
មុំដែលត្រូវនឹងតម្លៃនេះគឺ $\theta = \pi + \frac{\pi}{3} = \frac{4\pi}{3}$។
២០. ក្នុងប្លង់កុំផ្លិច សំណុំចំណុច $M$ មានកូអរដោនេដែលតំណាងដោយចំនួនកុំផ្លិច $z$ ផ្ទៀងផ្ទាត់សមីការ៖ $|z - 2 + i| = 3$ ជាអ្វី? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សរសេរសមីការឡើងវិញ៖ $|z - (2 - i)| = 3$។
នេះជាទម្រង់សមីការ $|z - z_I| = R$ ដែលតំណាងឱ្យរង្វង់ដែលមានផ្ចិត $I$ និងកាំ $R$។
ដូចនេះ ចំនួនកុំផ្លិច $z_I = 2 - i \implies$ ផ្ចិត $I(2, -1)$ និងកាំ $R = 3$។
២១. គណៈកម្មការមួយមានសមាជិក ៥ នាក់ ត្រូវជ្រើសរើសចេញពីបុរស ៦ នាក់ និងស្ត្រី ៤ នាក់។ រកចំនួនរបៀបក្នុងការជ្រើសរើស ដើម្បីឱ្យគណៈកម្មការនោះមានស្ត្រី ២ នាក់គត់។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ការជ្រើសរើសនេះប្រើ បន្សំ (Combination)៖
- ជ្រើសរើសស្ត្រី ២ នាក់ ពីចំណោម ៤ នាក់៖ $C(4, 2) = \frac{4 \times 3}{2 \times 1} = 6$ របៀប។
- ជ្រើសរើសបុរស ៣ នាក់ (ដើម្បីបង្គ្រប់ ៥នាក់) ពីចំណោម ៦ នាក់៖ $C(6, 3) = \frac{6 \times 5 \times 4}{3 \times 2 \times 1} = 20$ របៀប។
- តាមគោលការណ៍ផលគុណ ចំនួនរបៀបសរុបគឺ៖ $6 \times 20 = 120$ របៀប។
២២. គេបោះកាក់មួយដ៏ត្រឹមត្រូវចំនួន ៣ ដងព្រមគ្នា។ រកប្រូបាបដែលទទួលបានមុខរូប (H) យ៉ាងតិចណាស់មួយដង។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ចំនួនករណីអាចសរុប៖ $n(S) = 2^3 = 8$ ករណី។
- ប្រើព្រឹត្តិការណ៍ផ្ទុយ (គ្មានរូបទាល់តែសោះ គឺចេញផ្កា (T) ទាំង៣ដង)៖ មានតែ ១ ករណីគត់គឺ $(T, T, T)$។
- ប្រូបាបនៃព្រឹត្តិការណ៍ផ្ទុយ៖ $P(\bar{A}) = \frac{1}{8}$។
- ដូចនេះ ប្រូបាបទទួលបានមុខរូបយ៉ាងតិចមួយដងគឺ៖ $P(A) = 1 - P(\bar{A}) = 1 - \frac{1}{8} = \frac{7}{8}$។
២៣. រកកូអរដោនេកំណុំ $F$ នៃប៉ារ៉ាបូលដែលមានសមីការស្តង់ដា៖ $(y - 1)^2 = 8(x - 2)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការមានទម្រង់ស្តង់ដាអ័ក្សឆ្លាស់ដេក៖ $(y - k)^2 = 4p(x - h)$
- នាំឱ្យកំពូលមានកូអរដោនេ៖ $V(h, k) = V(2, 1)$។
- រកប៉ារ៉ាម៉ែត្រ $p$៖ $4p = 8 \implies p = 2$។
- ដោយសារអ័ក្សឆ្លាស់ដេក កំណុំមានរូបមន្ត៖ $F(h + p, k) = F(2 + 2, 1) = F(4, 1)$។
២៤. ក្នុងលំហកូអរដោនេ រកកូអរដោនេចំណុចប្រសព្វរវាងបន្ទាត់ $(L): \begin{cases} x = 1 + t \\ y = -1 + 2t \\ z = 2t \end{cases}$ និងប្លង់ $(P): 2x + y - z - 5 = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ជំនួសសមីការប៉ារ៉ាម៉ែត្រនៃបន្ទាត់ $(L)$ ទៅក្នុងសមីការប្លង់ $(P)$ ដើម្បីរកតម្លៃ $t$៖
$2(1 + t) + (-1 + 2t) - (2t) - 5 = 0$
$\implies 2 + 2t - 1 + 2t - 2t - 5 = 0$
$\implies 2t - 4 = 0 \implies t = 2$។
យកតម្លៃ $t = 2$ ទៅជំនួសក្នុងសមីការបន្ទាត់ឡើងវិញដើម្បីរកកូអរដោនេ៖
$x = 1 + 2 = 3$, $y = -1 + 2(2) = 3$, $z = 2(2) = 4$។
ដូចនេះចំណុចប្រសព្វគឺ $(3, 3, 4)$។
២៥. រកកូអរដោនេផ្ចិត $I$ និងប្រវែងកាំ $R$ នៃស៊្វែរដែលមានសមីការទូទៅ៖ $x^2 + y^2 + z^2 - 2x + 4y - 6z - 2 = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
បំលែងសមីការទូទៅទៅជាទម្រង់ស្តង់ដាដោយបង្គ្រប់ការេ៖
$(x^2 - 2x) + (y^2 + 4y) + (z^2 - 6z) = 2$
$\implies (x - 1)^2 - 1 + (y + 2)^2 - 4 + (z - 3)^2 - 9 = 2$
$\implies (x - 1)^2 + (y + 2)^2 + (z - 3)^2 = 2 + 1 + 4 + 9$
$\implies (x - 1)^2 + (y + 2)^2 + (z - 3)^2 = 16 = 4^2$។
នាំឱ្យបាន៖ ផ្ចិត $I(1, -2, 3)$ និងកាំ $R = \sqrt{16} = 4$។
