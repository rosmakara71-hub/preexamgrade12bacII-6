ពិន្ទុទទួលបាន៖ 0 / ១២៥

👇 សូមពិនិត្យវិធីសាស្ត្រដោះស្រាយលម្អិតនៅខាងក្រោមសំនួរនីមួយៗ!

១. គណនាលីមីត៖ $\lim_{x \to 3} \frac{x^2 - 9}{x^2 - 2x - 3}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ដាក់ភាគយក និងភាគបែងជាផលគុណកត្តា៖
- $x^2 - 9 = (x - 3)(x + 3)$
- $x^2 - 2x - 3 = (x - 3)(x + 1)$
$\lim_{x \to 3} \frac{(x-3)(x+3)}{(x-3)(x+1)} = \lim_{x \to 3} \frac{x+3}{x+1} = \frac{3+3}{3+1} = \frac{6}{4} = \frac{3}{2}$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{1 - \cos x}{x^2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តត្រីកោណមាត្រ $1 - \cos x = 2\sin^2\frac{x}{2}$៖
$\lim_{x \to 0} \frac{2\sin^2(x/2)}{x^2} = \lim_{x \to 0} 2 \cdot \left(\frac{\sin(x/2)}{2 \cdot (x/2)}\right)^2 = 2 \cdot \frac{1}{4} \cdot \lim_{x \to 0} \left(\frac{\sin(x/2)}{x/2}\right)^2 = \frac{2}{4} \cdot 1 = \frac{1}{2}$។
៣. គណនាលីមីត៖ $\lim_{x \to +\infty} \left(\sqrt{x^2 + 4x} - x\right)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $(+\infty - \infty)$។ គុណនិងចែកនឹងកន្សោមឆ្លាស់៖
$\lim_{x \to +\infty} \frac{(\sqrt{x^2+4x}-x)(\sqrt{x^2+4x}+x)}{\sqrt{x^2+4x}+x} = \lim_{x \to +\infty} \frac{x^2+4x-x^2}{x\sqrt{1+\frac{4}{x}}+x} = \lim_{x \to +\infty} \frac{4x}{x\left(\sqrt{1+\frac{4}{x}}+1\right)} = \frac{4}{1+1} = 2$។
៤. គណនាលីមីត៖ $\lim_{x \to 1} \frac{\ln x}{x - 1}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $t = x - 1 \implies x = 1 + t$។ កាលណា $x \to 1 \implies t \to 0$។
យើងបានលីមីតគំរូ៖ $\lim_{t \to 0} \frac{\ln(1+t)}{t} = 1$។
៥. រកដេរីវេនៃអនុគមន៍៖ $f(x) = e^{-x^2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេអនុគមន៍បណ្តាក់ $(e^u)' = u' \cdot e^u$ ដែលត្រង់នេះ $u = -x^2 \implies u' = -2x$៖
$f'(x) = (-x^2)' e^{-x^2} = -2xe^{-x^2}$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \frac{\ln x}{x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តផលចែក $\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$ ត្រង់នេះ $u = \ln x$ និង $v = x$៖
$f'(x) = \frac{(\ln x)' \cdot x - (\ln x) \cdot (x)'}{x^2} = \frac{\frac{1}{x} \cdot x - \ln x \cdot 1}{x^2} = \frac{1 - \ln x}{x^2}$ Blanco.
៧. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \cos\left(2x + \frac{\pi}{4}\right)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេ $(\cos u)' = -u' \cdot \sin u$ ដែលត្រង់នេះ $u = 2x + \frac{\pi}{4} \implies u' = 2$៖
$f'(x) = -\left(2x + \frac{\pi}{4}\right)' \sin\left(2x + \frac{\pi}{4}\right) = -2\sin\left(2x + \frac{\pi}{4}\right)$។
៨. រកចំណុចអតិបរមាធៀប (Local Maximum Point) នៃអនុគមន៍ $f(x) = x^3 - 3x$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- រកដេរីវេ៖ $f'(x) = 3x^2 - 3$
- ឱ្យ $f'(x) = 0 \implies 3x^2 = 3 \implies x = \pm 1$
- សិក្សាសញ្ញាដេរីវេទីពីរ៖ $f''(x) = 6x$
- ត្រង់ $x = -1 \implies f''(-1) = -6 < 0$ នាំឱ្យអនុគមន៍មានតម្លៃអតិបរមាធៀប។
- គណនាតម្លៃអតិបរមា៖ $f(-1) = (-1)^3 - 3(-1) = -1 + 3 = 2$។
ដូចនេះ ចំណុចអតិបរមាធៀបគឺ $(-1, 2)$។
៩. គណនាអាំងតេក្រាលមិនកំណត់៖ $\int (e^x + \cos x) \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមលក្ខណៈគ្រឹះនៃព្រីមីទីវ៖
$\int e^x \,dx = e^x + C$ និង $\int \cos x \,dx = \sin x + C$។
ដូចនេះ $\int (e^x + \cos x) \,dx = e^x + \sin x + C$។
១០. គណនាអាំងតេក្រាលកំណត់៖ $\int_{1}^{2} \frac{1}{2x + 1} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ព្រីមីទីវនៃ $\frac{1}{2x+1}$ គឺ $\frac{1}{2}\ln|2x+1|$។
$\int_{1}^{2} \frac{1}{2x+1} \,dx = \left[\frac{1}{2}\ln|2x+1|\right]_{1}^{2} = \frac{1}{2}\ln(5) - \frac{1}{2}\ln(3) = \frac{1}{2}(\ln 5 - \ln 3) = \frac{1}{2}\ln\frac{5}{3}$។
១១. គណនាអាំងតេក្រាលតាមផ្នែក៖ $\int_{0}^{\pi} x \sin x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = x \implies du = dx$ និង $dv = \sin x \,dx \implies v = -\cos x$។
តាមរូបមន្ត $\int u\,dv = uv - \int v\,du$៖
$\int_{0}^{\pi} x \sin x \,dx = \left[-x \cos x\right]_{0}^{\pi} - \int_{0}^{\pi} (-\cos x) \,dx$
$= \left(-\pi \cos\pi - 0\right) + \left[\sin x\right]_{0}^{\pi} = -\pi(-1) + (\sin\pi - \sin 0) = \pi + 0 = \pi$។
១២. គណនាផ្ទៃក្រឡាខណ្ឌដោយប៉ារ៉ាបូល $y = x^2$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x = 0$ ដល់ $x = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ផ្ទៃក្រឡា $S = \int_{0}^{3} x^2 \,dx = \left[\frac{x^3}{3}\right]_{0}^{3} = \frac{3^3}{3} - 0 = \frac{27}{3} = 9$ ឯកតាផ្ទៃ។
១៣. គណនាមាឌសូលីដបដិវត្តន៍បង្កើតដោយការបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយខ្សែគោង $y = \sqrt{x}$ និងបន្ទាត់ឈរ $x = 0$ ដល់ $x = 4$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តមាឌ៖ $V = \pi \int_{a}^{b} [f(x)]^2 \,dx$។
$V = \pi \int_{0}^{4} (\sqrt{x})^2 \,dx = \pi \int_{0}^{4} x \,dx = \pi \left[\frac{x^2}{2}\right]_{0}^{4} = \pi \left(\frac{16}{2} - 0\right) = 8\pi$ ឯកតាមាឌ។
១៤. រកចម្លើយពិសេសនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' - 2y = 0$ បើដឹងថា $y(0) = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $y' - 2y = 0 \implies y' = 2y$ មានចម្លើយទូទៅគឺ $y = Ke^{2x}$។
ជំនួសលក្ខខណ្ឌដើម $y(0) = 3 \implies 3 = Ke^{2(0)} \implies K = 3$។
ដូចនេះ ចម្លើយពិសេសគឺ $y = 3e^{2x}$។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' + 9y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការមានទម្រង់ $y'' + \omega^2 y = 0$ ដែលត្រង់នេះ $\omega^2 = 9 \implies \omega = 3$។
ដូចនេះ ចម្លើយទូទៅគឺ៖ $y = A\cos 3x + B\sin 3x$ ($A, B \in \mathbb{R}$)។
១៦. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' - 5y' + 6y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 - 5r + 6 = 0 \implies (r-2)(r-3) = 0 \implies r_1 = 2, r_2 = 3$។
ដោយមានឫសជាចំនួនពិតពីរផ្សេងគ្នា នាំឱ្យចម្លើយទូទៅគឺ $y = Ae^{2x} + Be^{3x}$។
១៧. គណនាតម្លៃនៃចំនួនកុំផ្លិច៖ $z = (1 + i)^4$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
យើងមាន $(1+i)^2 = 1 + 2i + i^2 = 1 + 2i - 1 = 2i$។
នាំឱ្យ $z = \left[(1+i)^2\right]^2 = (2i)^2 = 4i^2 = -4$ (ព្រោះ $i^2 = -1$)។
១៨. សរសេរចំនួនកុំផ្លិច $z = \sqrt{3} + i$ ជាទម្រង់ត្រីកោណមាត្រ។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ម៉ូឌុល $r = \sqrt{(\sqrt{3})^2 + 1^2} = \sqrt{3+1} = 2$។
- ទាញ $2$ ជាកត្តា៖ $z = 2\left(\frac{\sqrt{3}}{2} + i\frac{1}{2}\right)$។
- ដោយ $\cos\frac{\pi}{6} = \frac{\sqrt{3}}{2}$ និង $\sin\frac{\pi}{6} = \frac{1}{2}$ នាំឱ្យ $z = 2\left(\cos\frac{\pi}{6} + i\sin\frac{\pi}{6}\right)$។
១៩. រកចំនួនកុំផ្លិចឆ្លាស់ (Conjugate) $\bar{z}$ នៃចំនួនកុំផ្លិច $z = \frac{2}{1 + i}$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សម្រួល $z$ ជាមុនសិន៖ $z = \frac{2(1-i)}{(1+i)(1-i)} = \frac{2(1-i)}{1 - i^2} = \frac{2(1-i)}{2} = 1 - i$។
ដូចនេះ ចំនួនឆ្លាស់គឺ $\bar{z} = \overline{1-i} = 1 + i$។
២០. គណនាចំនួនបន្ទំ (Combination)៖ $C(5, 2)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តបន្ទំ $C(n, p) = \frac{n!}{p!(n-p)!}$៖
$C(5, 2) = \frac{5!}{2!(5-2)!} = \frac{5 \times 4 \times 3!}{2 \times 1 \times 3!} = \frac{20}{2} = 10$។
២១. ក្នុងថង់មួយមានឃ្លីពណ៌ក្រហម ៣ និងឃ្លីពណ៌ខៀវ ៤។ គេចាប់យកឃ្លី ២ ព្រមគ្នាចេញពីថង់ដោយចៃដន្យ។ រកប្រូបាបដើម្បីឱ្យចាប់បានឃ្លីពណ៌ក្រហមទាំង ២។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ចំនួនករណីអាច (ជ្រើសរើស ២ ពី ៧)៖ $n(S) = C(7, 2) = \frac{7 \times 6}{2} = 21$
- ចំនួនករណីស្រប (ជ្រើសរើស ២ ពីក្រហម ៣)៖ $n(A) = C(3, 2) = 3$
- ប្រូបាប៖ $P(A) = \frac{n(A)}{n(S)} = \frac{3}{21} = \frac{1}{7}$។
២២. រកកូអរដោនេកំនុំ (Focus) នៃប៉ារ៉ាបូលដែលមានសមីការ $x^2 = -12y$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការមានទម្រង់ស្តង់ដា $x^2 = 4py$ ដែលមានកំពូល $O(0,0)$ និងអ័ក្សឆ្លុះឈរ (អ័ក្ស $Oy$)។

ដូចនេះ កំនុំគឺ $F(0, p) = F(0, -3)$។
២៣. រកអុីសង់ទ្រីស៊ីតេ (Eccentricity) $e$ នៃអេលីបដែលមានសមីការ៖ $\frac{x^2}{25} + \frac{y^2}{9} = 1$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- យើងមាន $a^2 = 25 \implies a = 5$ និង $b^2 = 9 \implies b = 3$។
- គណនាចម្ងាយពីផ្ចិតទៅកំនុំ $c = \sqrt{a^2 - b^2} = \sqrt{25 - 9} = \sqrt{16} = 4$។
- រូបមន្តអុីសង់ទ្រីស៊ីតេ៖ $e = \frac{c}{a} = \frac{4}{5}$។
២៤. ក្នុងលំហកូអរដោនេ គណនាផលគុណស្កាលែររវាងពីរវ៉ិចទ័រ $\vec{u} = (1, -2, 2)$ និង $\vec{v} = (3, 0, -4)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រូបមន្តផលគុណស្កាលែរ៖ $\vec{u} \cdot \vec{v} = x_1x_2 + y_1y_2 + z_1z_2$។
$\vec{u} \cdot \vec{v} = (1)(3) + (-2)(0) + (2)(-4) = 3 + 0 - 8 = -5$។
២៥. រកសមីការប្លង់ដែលកាត់តាមចំណុច $A(1, 1, 1)$ ហើយមានវ៉ិចទ័រណរម៉ាល់ $\vec{n} = (2, 3, -1)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការប្លង់មានទម្រង់៖ $a(x - x_0) + b(y - y_0) + c(z - z_0) = 0$។
ជំនួសកូអរដោនេចូល៖ $2(x - 1) + 3(y - 1) - 1(z - 1) = 0$
$\implies 2x - 2 + 3y - 3 - z + 1 = 0 \implies 2x + 3y - z - 4 = 0$។
