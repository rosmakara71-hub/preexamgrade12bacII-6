👇 សូមពិនិត្យវិធីសាស្ត្រដោះស្រាយលម្អិតនៅខាងក្រោមសំនួរនីមួយៗ!

១. គណនាលីមីត៖ $\lim_{x \to 3} \frac{x^2 - 9}{x^2 - 2x - 3}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$ ចំពោះ $x \to 3$។
ដាក់ជាផលគុណកត្តា៖ ភាគយក $x^2 - 9 = (x - 3)(x + 3)$ ឯភាគបែង $x^2 - 2x - 3 = (x - 3)(x + 1)$។
$\lim_{x \to 3} \frac{(x-3)(x+3)}{(x-3)(x+1)} = \lim_{x \to 3} \frac{x+3}{x+1} = \frac{3+3}{3+1} = \frac{6}{4} = \frac{3}{2}$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{1 - \cos 2x}{x^2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តត្រីកោណមាត្រ៖ $1 - \cos 2x = 2\sin^2 x$។
$\lim_{x \to 0} \frac{2\sin^2 x}{x^2} = 2 \times \left(\lim_{x \to 0} \frac{\sin x}{x}\right)^2 = 2 \times (1)^2 = 2$។
៣. គណនាលីមីត៖ $\lim_{x \to +\infty} \left(1 + \frac{2}{x}\right)^x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $1^\infty$។ ប្រើរូបមន្តគំរូ $\lim_{u \to \pm\infty} \left(1 + \frac{1}{u}\right)^u = e$។
តាង $u = \frac{x}{2} \implies x = 2u$។ កាលណា $x \to +\infty \implies u \to +\infty$។
$\lim_{u \to +\infty} \left(1 + \frac{1}{u}\right)^{2u} = \left[\lim_{u \to +\infty} \left(1 + \frac{1}{u}\right)^u\right]^2 = e^2$។
៤. រកតម្លៃ $k$ ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} \frac{\sin 4x}{2x} & \text{បើ } x \neq 0 \\ k & \text{បើ } x = 0 \end{cases}$ ជាប់ត្រង់ $x = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
អនុគមន៍ជាប់ត្រង់ $x=0$ លុះត្រាតែ $\lim_{x \to 0} f(x) = f(0) = k$។
គណនា៖ $\lim_{x \to 0} \frac{\sin 4x}{2x} = \lim_{x \to 0} \left( \frac{\sin 4x}{4x} \times \frac{4x}{2x} \right) = 1 \times 2 = 2$។
ដូចនេះ $k = 2$។
៥. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \ln(x^2 + 1)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេនៃអនុគមន៍លោការីតបណ្តាក់ $(\ln u)' = \frac{u'}{u}$។
នៅទីនេះ $u = x^2 + 1 \implies u' = (x^2 + 1)' = 2x$។
នាំឱ្យបាន $f'(x) = \frac{2x}{x^2 + 1}$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = e^{3x} \cos x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេផលគុណ $(uv)' = u'v + uv'$ និងរូបមន្ត $(e^{ax})' = a \cdot e^{ax}$។
$f'(x) = (e^{3x})'\cos x + e^{3x}(\cos x)' = 3e^{3x}\cos x + e^{3x}(-\sin x)$
$= e^{3x}(3\cos x - \sin x)$។
៧. រកសមីការបន្ទាត់ប៉ះខ្សែគោង $y = x^2 - 3x + 2$ ត្រង់ចំណុចដែលមានអាប់ស៊ីស $x = 1$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការបន្ទាត់ប៉ះគឺ $y = f'(1)(x - 1) + f(1)$។
- រកអរដោនេ $f(1) = 1^2 - 3(1) + 2 = 1 - 3 + 2 = 0$។
- រកមេគុណប្រាប់ទិស $f'(x) = 2x - 3 \implies f'(1) = 2(1) - 3 = -1$។
ជំនួសក្នុងសមីការ៖ $y = -1(x - 1) + 0 \implies y = -x + 1$ biographies។
៨. រកតម្លៃអប្បបរមាក្នុងស្រុករបស់អនុគមន៍ $f(x) = x^3 - 3x$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- រកដេរីវេ៖ $f'(x) = 3x^2 - 3$។
- ឱ្យ $f'(x) = 0 \implies 3(x^2 - 1) = 0 \implies x = \pm 1$។
- តាមតារាងអថេរភាព អនុគមន៍មានអប្បបរមាត្រង់ $x = 1$។
- គណនាតម្លៃអប្បបរមា៖ $f(1) = 1^3 - 3(1) = 1 - 3 = -2$។
៩. គណនាអាំងតេក្រាលមិនកំណត់៖ $\int \cos(5x - 1) \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តគ្រឹះ $\int \cos(ax+b) \,dx = \frac{1}{a}\sin(ax+b) + C$។
នៅទីនេះ $a = 5$ នាំឱ្យបាន៖ $\int \cos(5x - 1) \,dx = \frac{1}{5}\sin(5x - 1) + C$។
១០. គណនាអាំងតេក្រាលកំណត់៖ $\int_{1}^{e} \frac{1}{x} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ព្រីមីទីវនៃ $\frac{1}{x}$ គឺ $\ln|x|$។
$\int_{1}^{e} \frac{1}{x} \,dx = \left[\ln x\right]_{1}^{e} = \ln e - \ln 1 = 1 - 0 = 1$។
១១. គណនាអាំងតេក្រាលកំណត់៖ $\int_{0}^{\pi} x \sin x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើអាំងតេក្រាលដោយផ្នែក $\int u \,dv = uv - \int v \,du$៖
តាង $u = x \implies du = dx$
តាង $dv = \sin x \,dx \implies v = -\cos x$
$\int_{0}^{\pi} x \sin x \,dx = \left[-x\cos x\right]_{0}^{\pi} - \int_{0}^{\pi} (-\cos x) \,dx$
$= \Big(-\pi\cos\pi - (-0)\Big) + \int_{0}^{\pi} \cos x \,dx = -\pi(-1) + \left[\sin x\right]_{0}^{\pi}$
$= \pi + (\sin\pi - \sin0) = \pi + 0 = \pi$។
១២. គណនាផ្ទៃក្រឡាខណ្ឌដោយខ្សែគោង $y = x^2$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x = 0$ ដល់ $x = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ផ្ទៃក្រឡា៖ $S = \int_{0}^{3} x^2 \,dx = \left[\frac{x^3}{3}\right]_{0}^{3} = \frac{3^3}{3} - 0 = \frac{27}{3} = 9$ ឯកតាផ្ទៃ។
១៣. គណនាមាឌសូលីដបដិវត្តន៍បង្កើតឡើងដោយការបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយខ្សែគោង $y = \sqrt{x}$ និងបន្ទាត់ឈរ $x = 0$ ដល់ $x = 4$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖

១៤. រកចម្លើយពិសេសនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' - 2y = 0$ បើដឹងថា $y(0) = 5$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $y' - 2y = 0 \implies y' = 2y$ មានចម្លើយទូទៅ៖ $y = Ke^{2x}$។
តាមលក្ខខណ្ឌដើម $y(0) = 5 \implies 5 = Ke^{2(0)} = K \cdot 1 \implies K = 5$។
ដូចនេះ ចម្លើយពិសេសគឺ $y = 5e^{2x}$។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' - 5y' + 6y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈគឺ៖ $r^2 - 5r + 6 = 0$។
តាមដីគ្រីមីណង់ ឬតាមផលបូកផលគុណឫស៖ $(r - 2)(r - 3) = 0 \implies r_1 = 2, r_2 = 3$។
ដោយសារមានឫសពីរផ្សេងគ្នាជាចំនួនពិត នាំឱ្យចម្លើយទូទៅគឺ៖ $y = Ae^{2x} + Be^{3x}$ ($A, B \in \mathbb{R}$)។
១៦. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' + 9y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 + 9 = 0 \implies r^2 = -9 \implies r = \pm 3i$ (ឫសកុំផ្លិចឆ្លាស់ $\alpha = 0, \beta = 3$)។
ចម្លើយទូទៅមានទម្រង់៖ $y = e^{\alpha x}(A\cos\beta x + B\sin\beta x)$

១៧. គណនាផលគុណចំនួនកុំផ្លិច៖ $z = (1 + 2i)(3 - i)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ពង្រាតផលគុណតាមធម្មតា៖

១៨. សរសេរចំនួនកុំផ្លិច $z = 1 + i\sqrt{3}$ ជាទម្រង់ត្រីកោណមាត្រ។ (៥ ពិន្ទុ)

១៩. ក្នុងប្លង់កុំផ្លិច សំណុំចំណុច $M(z)$ ដែលផ្ទៀងផ្ទាត់លក្ខខណ្ឌ $|z + i| = |z - 1|$ គឺជាអ្វី? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $|z - z_A| = |z - z_B|$ គឺជាសំណុំចំណុចស្ថិតនៅចម្ងាយស្មើគ្នាពីចំណុច $A$ និង $B$ ពោលគឺបន្ទាត់មេដ្យាទ័រនៃអង្កត់ $[AB]$។
នៅទីនេះ $z_A = -i \implies A(0, -1)$ និង $z_B = 1 \implies B(1, 0)$។
ដូចនេះ វាជាមេដ្យាទ័រនៃអង្កត់ភ្ជាប់ចំណុចទាំងពីរនេះ។
២០. គេចង់ជ្រើសរើសសិស្ស ៣ នាក់ចេញពីសិស្សទាំង ៧ នាក់ ដើម្បីបង្កើតជាគណៈកម្មការមួយ។ តើគេអាចជ្រើសរើសបានប៉ុន្មានរបៀប? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ការជ្រើសរើសមនុស្ស ៣ នាក់ក្នុងចំណោម ៧ នាក់ដោយមិនគិតលំដាប់ គឺជាបន្ទំ (Combination)៖
$C(7,3) = \frac{7!}{3!(7-3)!} = \frac{7 \times 6 \times 5}{3 \times 2 \times 1} = 7 \times 5 = 35$ របៀប។
២១. ក្នុងថង់មួយមានឃ្លីពណ៌ក្រហម ៤ និងឃ្លីពណ៌ខៀវ ៦។ គេចាប់យកឃ្លី ២ ព្រមគ្នាចេញពីថង់ដោយចៃដន្យ។ រកប្រូបាបដើម្បីឱ្យចាប់បានឃ្លីពណ៌ក្រហមទាំងពីរ។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖

២២. រកសមីការស្តង់ដានៃប៉ារ៉ាបូលដែលមានកំពូលនៅគល់កូអរដោនេ $O(0,0)$ និងមានបន្ទាត់ប្រាប់ទិស $y = -3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
បន្ទាត់ប្រាប់ទិសមានសមីការ $y = -p = -3 \implies p = 3 > 0$។
ដោយសារបន្ទាត់ប្រាប់ទិសជាបន្ទាត់ដេក នាំឱ្យប៉ារ៉ាបូលមានអ័ក្សឆ្លុះឈរ (បើងាកឡើងលើ) សមីការស្តង់ដាគឺ៖ $x^2 = 4py$។
ជំនួស $p = 3$ ចូល៖ $x^2 = 4(3)y \implies x^2 = 12y$។
២៣. រកអុិចសង់ទ្រីស៊ីតេ (Eccentricity) នៃអេលីបដែលមានសមីការ៖ $\frac{x^2}{25} + \frac{y^2}{16} = 1$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមសមីការអេលីប៖ $a^2 = 25 \implies a = 5$ និង $b^2 = 16 \implies b = 4$។
- គណនាចម្ងាយផ្ចិតទៅកំនុំ៖ $c = \sqrt{a^2 - b^2} = \sqrt{25 - 16} = \sqrt{9} = 3$។
- រូបមន្តអុិចសង់ទ្រីស៊ីតេ៖ $e = \frac{c}{a} = \frac{3}{5} = 0.6$។
២៤. ក្នុងលំហកូអរដោនេ គេមានវ៉ិចទ័រ $\vec{u} = (2, -1, 3)$ និង $\vec{v} = (1, 4, 1)$។ គណនាផលគុណស្កាលែ $\vec{u} \cdot \vec{v}$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តផលគុណស្កាលែក្នុងលំហ៖ $\vec{u} \cdot \vec{v} = x_1x_2 + y_1y_2 + z_1z_2$។
$\vec{u} \cdot \vec{v} = 2(1) + (-1)(4) + 3(1) = 2 - 4 + 3 = 1$។
