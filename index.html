- $\theta = \theta_1 + \theta_2 = \frac{\pi}{4} + \frac{\pi}{12} = \frac{3\pi + \pi}{12} = \frac{4\pi}{12} = \frac{\pi}{3}$។
ដូចនេះ $z_1 \times z_2 = 6\left(\cos\frac{\pi}{3} + i\sin\frac{\pi}{3}\right)$។
