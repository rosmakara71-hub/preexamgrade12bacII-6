ពិន្ទុទទួលបាន៖ 0 / ១២៥

👇 សូមពិនិត្យវិធីសាស្ត្រដោះស្រាយលម្អិតនៅខាងក្រោមសំនួរនីមួយៗ!

១. គណនាលីមីត៖ $\lim_{x \to 2} \frac{x^3 - 8}{x - 2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ប្រើរូបមន្តផលដកគូប $a^3 - b^3 = (a-b)(a^2 + ab + b^2)$៖
$\lim_{x \to 2} \frac{(x-2)(x^2 + 2x + 4)}{x-2} = \lim_{x \to 2} (x^2 + 2x + 4)$
$= 2^2 + 2(2) + 4 = 4 + 4 + 4 = 12$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{\tan 3x}{\sin 5x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ចែកភាគយកនិងភាគបែងនឹង $x$ រួចប្រើលីមីតគំរូ $\lim_{u \to 0} \frac{\sin u}{u} = 1$ និង $\lim_{u \to 0} \frac{\tan u}{u} = 1$៖
$\lim_{x \to 0} \frac{\frac{\tan 3x}{x}}{\frac{\sin 5x}{x}} = \frac{\lim_{x \to 0} \frac{\tan 3x}{3x} \times 3}{\lim_{x \to 0} \frac{\sin 5x}{5x} \times 5} = \frac{1 \times 3}{1 \times 5} = \frac{3}{5}$។
៣. គណនាលីមីត៖ $\lim_{x \to 0} (1 - 3x)^{\frac{1}{x}}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $1^\infty$។ ប្រើរូបមន្តគ្រឹះ $\lim_{u \to 0} (1 + u)^{\frac{1}{u}} = e$។
សរសេរលីមីតឡើងវិញ៖ $\lim_{x \to 0} \left[ (1 + (-3x))^{\frac{1}{-3x}} \right]^{-3} = e^{-3}$។
៤. រកតម្លៃ $a$ ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} \frac{\sqrt{x+1} - 1}{x} & \text{បើ } x \neq 0 \\ a & \text{បើ } x = 0 \end{cases}$ ជាប់ត្រង់ $x = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ដើម្បីឱ្យ $f$ ជាប់ត្រង់ $x=0$ លុះត្រាតែ $\lim_{x \to 0} f(x) = f(0) = a$។
គណនាលីមីតដោយគុណកន្សោមឆ្លាស់៖
$\lim_{x \to 0} \frac{(\sqrt{x+1}-1)(\sqrt{x+1}+1)}{x(\sqrt{x+1}+1)} = \lim_{x \to 0} \frac{x+1-1}{x(\sqrt{x+1}+1)} = \lim_{x \to 0} \frac{1}{\sqrt{x+1}+1} = \frac{1}{1+1} = \frac{1}{2}$។
ដូចនេះ $a = \frac{1}{2}$។
៥. រកដេរីវេនៃអនុគមន៍៖ $f(x) = e^{2x} \cos x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេផលគុណ $(uv)' = u'v + uv'$ ដែល $u=e^{2x} \implies u'=2e^{2x}$ និង $v=\cos x \implies v'=-\sin x$។
$f'(x) = (2e^{2x})\cos x + e^{2x}(-\sin x) = e^{2x}(2\cos x - \sin x)$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \ln(x^2 + 1)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេ $(\ln u)' = \frac{u'}{u}$ ដែល $u = x^2 + 1 \implies u' = 2x$។
នាំឱ្យបាន៖ $f'(x) = \frac{2x}{x^2+1}$។
៧. រកសមីការបន្ទាត់ប៉ះខ្សែគោង $y = \frac{1}{x}$ ត្រង់ចំណុចដែលមានអាប់ស៊ីស $x = 1$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការបន្ទាត់ប៉ះ៖ $y = f'(1)(x - 1) + f(1)$
- គណនា $f(1) = \frac{1}{1} = 1$។
- រកដេរីវេ $f'(x) = -\frac{1}{x^2} \implies f'(1) = -\frac{1}{1^2} = -1$។
នាំឱ្យបានសមីការ៖ $y = -1(x - 1) + 1 = -x + 1 + 1 = -x + 2$។
៨. រកចំណុចអតិបរមារបស់អនុគមន៍៖ $f(x) = \frac{1}{3}x^3 - x^2 - 3x + 1$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គណនាដេរីវេ៖ $f'(x) = x^2 - 2x - 3$។
ឱ្យ $f'(x) = 0 \implies (x-3)(x+1) = 0 \implies x = 3$ ឬ $x = -1$។
តាមតារាងសញ្ញា $f'(x)$៖ ដេរីវេប្តូរសញ្ញាពីវិជ្ជមាន (+) ទៅអវិជ្ជមាន (-) ត្រង់ចំណុច $x = -1$ ដូចនេះអនុគមន៍មានតម្លៃអតិបរមាត្រង់ $x = -1$។
៩. គណនាអាំងតេក្រាលដោយផ្នែក៖ $\int x \cos x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = x \implies du = dx$ និង $dv = \cos x dx \implies v = \sin x$។
តាមរូបមន្ត $\int u dv = uv - \int v du$ ៖
$\int x \cos x dx = x\sin x - \int \sin x dx = x\sin x - (-\cos x) + C = x\sin x + \cos x + C$។
១០. គណនាអាំងតេក្រាលកំណត់៖ $\int_{1}^{2} \frac{1}{x^2} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ព្រីមីទីវនៃ $\frac{1}{x^2}$ គឺ $-\frac{1}{x}$។
$\int_{1}^{2} \frac{1}{x^2} dx = \left[-\frac{1}{x}\right]_{1}^{2} = \left(-\frac{1}{2}\right) - \left(-\frac{1}{1}\right) = -\frac{1}{2} + 1 = \frac{1}{2}$។
១១. គណនាអាំងតេក្រាលកំណត់៖ $\int_{0}^{\ln 2} e^{2x} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ព្រីមីទីវនៃ $e^{2x}$ គឺ $\frac{1}{2}e^{2x}$។
$\int_{0}^{\ln 2} e^{2x} dx = \left[\frac{1}{2}e^{2x}\right]_{0}^{\ln 2} = \frac{1}{2}(e^{2\ln 2} - e^0)$
$= \frac{1}{2}(e^{\ln 4} - 1) = \frac{1}{2}(4 - 1) = \frac{3}{2}$។
១២. គណនាផ្ទៃក្រឡាខណ្ឌដោយខ្សែគោង $y = e^x$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x = 0$ ដល់ $x = \ln 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ផ្ទៃក្រឡា៖ $S = \int_{0}^{\ln 3} e^x dx = [e^x]_{0}^{\ln 3} = e^{\ln 3} - e^0 = 3 - 1 = 2$ ឯកតាផ្ទៃ។
១៣. គណនាមាឌសូលីដបដិវត្តន៍បង្កើតឡើងដោយបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយបន្ទាត់ $y = x$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x=0$ ដល់ $x=3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តមាឌ៖ $V = \pi \int_{0}^{3} [f(x)]^2 dx = \pi \int_{0}^{3} x^2 dx$។
$= \pi \left[\frac{x^3}{3}\right]_{0}^{3} = \pi \left(\frac{27}{3} - 0\right) = 9\pi$ ឯកតាមាឌ។
១៤. រកចម្លើយពិសេសនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' + 3y = 0$ បើដឹងថា $y(0) = 2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $y' + 3y = 0 \implies y' = -3y$ មានចម្លើយទូទៅ៖ $y = Ke^{-3x}$។
ប្រើលក្ខខណ្ឌដើម $y(0) = 2 \implies 2 = Ke^{-3(0)} \implies K = 2$។
ដូចនេះ ចម្លើយពិសេសគឺ $y = 2e^{-3x}$។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' - 4y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 - 4 = 0 \implies r^2 = 4 \implies r = \pm 2$ (មានឫសពិតពីរផ្សេងគ្នា $r_1 = 2, r_2 = -2$)។
ដូចនេះ ចម្លើយទូទៅគឺ៖ $y = Ae^{2x} + Be^{-2x}$។
១៦. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' - y = 2$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ទម្រង់សមីការ $y' + ay = b$ ដែល $a = -1, b = 2$។
រូបមន្តចម្លើយទូទៅ៖ $y = Ke^{-ax} + \frac{b}{a} = Ke^{-(-1)x} + \frac{2}{-1} = Ke^x - 2$។
១៧. រកម៉ូឌុលនៃចំនួនកុំផ្លិច៖ $z = 3 + 4i$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តម៉ូឌុល៖ $|z| = \sqrt{a^2 + b^2}$
$|z| = \sqrt{3^2 + 4^2} = \sqrt{9 + 16} = \sqrt{25} = 5$។
១៨. គណនាតម្លៃផលចែកចំនួនកុំផ្លិច៖ $z = \frac{1+i}{1-i}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គុណភាគយកនិងភាគបែងនឹងកន្សោមឆ្លាស់នៃភាគបែង $(1+i)$៖
$z = \frac{(1+i)(1+i)}{(1-i)(1+i)} = \frac{1 + 2i + i^2}{1^2 - i^2} = \frac{1 + 2i - 1}{1 - (-1)} = \frac{2i}{2} = i$។
១៩. សរសេរចំនួនកុំផ្លិច $z = 1 - i$ ជាទម្រង់អិចស្ប៉ូណង់ស្យែល។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ម៉ូឌុល៖ $r = \sqrt{1^2 + (-1)^2} = \sqrt{2}$។
- អាគុយម៉ង់៖ $\cos\theta = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$ និង $\sin\theta = -\frac{1}{\sqrt{2}} = -\frac{\sqrt{2}}{2} \implies \theta = -\frac{\pi}{4}$។
ដូចនេះ ទម្រង់អិចស្ប៉ូណង់ស្យែលគឺ៖ $z = r e^{i\theta} = \sqrt{2}e^{-i\frac{\pi}{4}}$។
២០. ក្នុងប្លង់កុំផ្លិច សំណុំចំណុច $M(z)$ ដែលផ្ទៀងផ្ទាត់លក្ខខណ្ឌ $|z - 2 + 3i| = 4$ គឺជាអ្វី? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $|z - (2 - 3i)| = 4$ អាចសរសេរជា $|z - z_0| = R$ ដែល $z_0 = 2 - 3i$ និង $R = 4$។
ដូចនេះ វាជាសមីការរង្វង់ដែលមានផ្ចិតមានកូអរដោនេ $(2, -3)$ និងកាំស្មើ $4$។
២១. តើគេអាចរៀបចំសៀវភៅផ្សេងៗគ្នា ចំនួន ៤ ក្បាល តម្រៀបជាជួរនៅលើធ្នើរបានប៉ុន្មានរបៀប? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ការតម្រៀបវត្ថុ $n$ ខុសៗគ្នាជាជួរ គឺជាលំដាប់ (Permutation) ដែលប្រើរូបមន្ត $n!$ (ណ factorial)៖
ចំនួនរបៀប = $4! = 4 \times 3 \times 2 \times 1 = 24$ របៀប។
២២. គេបោះគ្រាប់ឡុកឡាក់ ១ គ្រាប់ដែលត្រឹមត្រូវ។ រកប្រូបាបដើម្បីបោះបានលេខជា ចំនួនបឋម។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ចំនួនករណីអាចសរុប៖ $S = \{1, 2, 3, 4, 5, 6\} \implies n(S) = 6$។
- ករណីស្រប (លេខជាចំនួនបឋម)៖ $A = \{2, 3, 5\} \implies n(A) = 3$។
- ប្រូបាប៖ $P(A) = \frac{n(A)}{n(S)} = \frac{3}{6} = \frac{1}{2}$។
២៣. រកសមីការស្តង់ដានៃប៉ារ៉ាបូលដែលមានកំពូល $O(0,0)$ និងចំណុចកំនុំ $F(0,2)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ដោយសារកំពូល $O(0,0)$ ហើយកំនុំ $F(0,2)$ ស្ថិតនៅលើអ័ក្សអរដោនេ ($Oy$) នាំឱ្យអ័ក្សឆ្លុះជាអ័ក្សឈរ។
សមីការស្តង់ដាមានទម្រង់ $x^2 = 4py$ ដែល $p = 2$ (ចម្ងាយពីកំពូលទៅកំនុំ)។
ដូចនេះ សមីការគឺ $x^2 = 4(2)y \implies x^2 = 8y$។
២៤. ក្នុងលំហកូអរដោនេ គណនាផលគុណស្កាលែ $\vec{u} \cdot \vec{v}$ បើដឹងថា $\vec{u} = (2, -1, 3)$ និង $\vec{v} = (1, 4, 2)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តផលគុណស្កាលែ៖ $\vec{u} \cdot \vec{v} = x x' + y y' + z z'$
$\vec{u} \cdot \vec{v} = 2(1) + (-1)(4) + 3(2) = 2 - 4 + 6 = 4$។
២៥. រកសមីការស៊្វែរដែលមានផ្ចិត $I(1, 0, -2)$ និងកាំ $R = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការស្តង់ដានៃស៊្វែរ៖ $(x-a)^2 + (y-b)^2 + (z-c)^2 = R^2$ ដែលមានផ្ចិត $I(a, b, c)$។
ជំនួសកូអរដោនេផ្ចិត $a=1, b=0, c=-2$ និង $R=3$ ៖
$(x-1)^2 + (y-0)^2 + (z-(-2))^2 = 3^2 \implies (x-1)^2 + y^2 + (z+2)^2 = 9$។
