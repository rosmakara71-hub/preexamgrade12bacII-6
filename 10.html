១. គណនាលីមីត៖ $\lim_{x \to 3} \frac{x^2 - 9}{\sqrt{x+1} - 2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ គុណកន្សោមឆ្លាស់នៃភាគបែងគឺ $\sqrt{x+1}+2$៖
$\lim_{x \to 3} \frac{(x-3)(x+3)(\sqrt{x+1}+2)}{x+1-4} = \lim_{x \to 3} \frac{(x-3)(x+3)(\sqrt{x+1}+2)}{x-3}$
$= \lim_{x \to 3} (x+3)(\sqrt{x+1}+2) = (3+3)(\sqrt{3+1}+2) = 6 \times (2+2) = 24$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{1 - \cos 4x}{x^2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តត្រីកោណមាត្រ៖ $1 - \cos 4x = 2\sin^2 2x$។
$\lim_{x \to 0} \frac{2\sin^2 2x}{x^2} = 2 \times \lim_{x \to 0} \left(\frac{\sin 2x}{2x} \times 2\right)^2 = 2 \times (1 \times 2)^2 = 2 \times 4 = 8$។
៣. គណនាលីមីត៖ $\lim_{x \to +\infty} \left(1 + \frac{2}{x}\right)^x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $1^\infty$។ ប្រើរូបមន្តគ្រឹះ $\lim_{u \to \pm\infty} \left(1 + \frac{1}{u}\right)^u = e$។
តាង $u = \frac{x}{2} \implies x = 2u$។ កាលណា $x \to +\infty \implies u \to +\infty$។
នាំឱ្យបាន៖ $\lim_{u \to +\infty} \left(1 + \frac{1}{u}\right)^{2u} = \left[\lim_{u \to +\infty} \left(1 + \frac{1}{u}\right)^u\right]^2 = e^2$។
៤. រកតម្លៃ $m$ ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} \frac{\sin 2x}{x} & \text{បើ } x \neq 0 \\ m & \text{បើ } x = 0 \end{cases}$ ជាប់ត្រង់ $x = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
អនុគមន៍ $f$ ជាប់ត្រង់ $x=0$ លុះត្រាតែ $\lim_{x \to 0} f(x) = f(0)$។
គណនាលីមីត៖ $\lim_{x \to 0} \frac{\sin 2x}{x} = \lim_{x \to 0} \frac{\sin 2x}{2x} \times 2 = 1 \times 2 = 2$។
ដូចនេះ $f(0) = m = 2$។
៥. រកដេរីវេនៃអនុគមន៍៖ $f(x) = x \ln x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេផលគុណ $(uv)' = u'v + uv'$ ដែល $u=x \implies u'=1$ និង $v=\ln x \implies v'=\frac{1}{x}$។
$f'(x) = (1)\ln x + x\left(\frac{1}{x}\right) = \ln x + 1$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \sqrt{x^2 + 3}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេអនុគមន៍បណ្តាក់ $(\sqrt{u})' = \frac{u'}{2\sqrt{u}}$ ដែល $u = x^2 + 3 \implies u' = 2x$។
$f'(x) = \frac{2x}{2\sqrt{x^2+3}} = \frac{x}{\sqrt{x^2+3}}$។
៧. រកសមីការបន្ទាត់ប៉ះខ្សែគោង $y = e^x$ ត្រង់ចំណុចដែលមានអាប់ស៊ីស $x = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការបន្ទាត់ប៉ះ៖ $y = f'(0)(x - 0) + f(0)$
- តម្លៃអនុគមន៍៖ $f(0) = e^0 = 1$។
- មេគុណប្រាប់ទិស៖ $f'(x) = e^x \implies f'(0) = e^0 = 1$។
នាំឱ្យបានសមីការ៖ $y = 1(x - 0) + 1 \implies y = x + 1$។
៨. រកចំណុចអប្បបរមារបស់អនុគមន៍៖ $f(x) = x^3 - 3x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
គណនាដេរីវេ៖ $f'(x) = 3x^2 - 3$។ ឱ្យ $f'(x) = 0 \implies 3(x^2 - 1) = 0 \implies x = \pm 1$។
គណនាដេរីវេទីពីរ៖ $f''(x) = 6x$៖
- ត្រង់ $x = 1 \implies f''(1) = 6 > 0 \implies$ អនុគមន៍មានតម្លៃអប្បបរមាត្រង់ $x = 1$។
- ត្រង់ $x = -1 \implies f''(-1) = -6 < 0 \implies$ អនុគមន៍មានតម្លៃអតិបរមាត្រង់ $x = -1$។
៩. គណនាអាំងតេក្រាលមិនកំណត់៖ $\int \frac{1}{2x+3} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តព្រីមីទីវ៖ $\int \frac{u'}{u} dx = \ln|u| + C$។
ទីនេះ $u = 2x+3 \implies u' = 2$ នាំឱ្យសរសេរ៖ $\int \frac{1}{2x+3} dx = \frac{1}{2} \int \frac{2}{2x+3} dx = \frac{1}{2}\ln|2x+3| + C$។
១០. គណនាអាំងតេក្រាលកំណត់៖ $\int_{0}^{\frac{\pi}{2}} \sin x \cos^2 x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = \cos x \implies du = -\sin x dx \implies \sin x dx = -du$។
ប្តូរគោល៖ បើ $x = 0 \implies u = \cos 0 = 1$ ៖ បើ $x = \frac{\pi}{2} \implies u = \cos\frac{\pi}{2} = 0$។
នាំឱ្យបាន៖ $\int_{1}^{0} u^2 (-du) = \int_{0}^{1} u^2 du = \left[\frac{u^3}{3}\right]_{0}^{1} = \frac{1}{3} - 0 = \frac{1}{3}$។
១១. គណនាអាំងតេក្រាលកំណត់ដោយផ្នែក៖ $\int_{0}^{1} x e^x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = x \implies du = dx$ និង $dv = e^x dx \implies v = e^x$។
តាមរូបមន្ត $\int u dv = uv - \int v du$ ៖
$\int_{0}^{1} x e^x dx = [x e^x]_{0}^{1} - \int_{0}^{1} e^x dx = (1e^1 - 0) - [e^x]_{0}^{1} = e - (e^1 - e^0) = e - e + 1 = 1$។
១២. គណនាផ្ទៃក្រឡាខណ្ឌដោយខ្សែគោង $y = x^2$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x = 1$, $x = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ផ្ទៃក្រឡា៖ $S = \int_{1}^{3} x^2 dx = \left[\frac{x^3}{3}\right]_{1}^{3} = \frac{3^3}{3} - \frac{1^3}{3} = \frac{27}{3} - \frac{1}{3} = \frac{26}{3}$ ឯកតាផ្ទៃ។
១៣. គណនាមាឌសូលីដបដិវត្តន៍បង្កើតឡើងដោយបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយខ្សែគោង $y = \sqrt{x}$ បន្ទាត់ឈរ $x=0$ និង $x=4$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តមាឌ៖ $V = \pi \int_{0}^{4} [f(x)]^2 dx = \pi \int_{0}^{4} (\sqrt{x})^2 dx = \pi \int_{0}^{4} x dx$។
$= \pi \left[\frac{x^2}{2}\right]_{0}^{4} = \pi \left(\frac{16}{2} - 0\right) = 8\pi$ ឯកតាមាឌ។
១៤. រកចម្លើយពិសេសនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' - 2y = 0$ បើដឹងថា $y(0) = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការមានទម្រង់ $y' - ay = 0$ មានចម្លើយទូទៅគឺ $y = Ke^{ax}$ ដែល $a = 2 \implies y = Ke^{2x}$។
ប្រើលក្ខខណ្ឌដើម $y(0) = 3 \implies 3 = Ke^{2(0)} \implies K = 3$។
ដូចនេះ ចម្លើយពិសេសគឺ $y = 3e^{2x}$។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' + 9y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 + 9 = 0 \implies r^2 = -9 \implies r = \pm 3i$ (ឫសកុំផ្លិចសុទ្ធ គ្មានផ្នែកពិត $\alpha = 0, \beta = 3$)។
ចម្លើយទូទៅគឺ៖ $y = e^{0x}(A\cos 3x + B\sin 3x) = A\cos 3x + B\sin 3x$។
១៦. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' + y = 1$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ទម្រង់សមីការគឺ $y' + ay = b$ ដែល $a = 1, b = 1$។
រូបមន្តចម្លើយទូទៅ៖ $y = Ke^{-ax} + \frac{b}{a} = Ke^{-1x} + \frac{1}{1} = Ke^{-x} + 1$។
១៧. គណនាតម្លៃនៃចំនួនកុំផ្លិច៖ $z = (1 + i)^2$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
លាតតាមរូបមន្តដឺក្រេទី២៖ $(a+b)^2 = a^2 + 2ab + b^2$
$z = 1^2 + 2(1)(i) + i^2 = 1 + 2i - 1 = 2i$ (ដោយសារ $i^2 = -1$)។
១៨. រកចំនួនកុំផ្លិចឆ្លាស់នៃ $z = \frac{1}{i}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
បំប្លែង $z$ ជាមុនសិន៖ $z = \frac{1}{i} = \frac{1 \times (-i)}{i \times (-i)} = \frac{-i}{-i^2} = \frac{-i}{-(-1)} = -i$។
ដូចនេះ ចំនួនកុំផ្លិចឆ្លាស់នៃ $z = -i$ គឺ $\bar{z} = i$។
១៩. រកអាគុយម៉ង់នៃចំនួនកុំផ្លិច៖ $z = -\sqrt{3} + i$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ម៉ូឌុល៖ $r = \sqrt{(-\sqrt{3})^2 + 1^2} = \sqrt{3+1} = 2$។
- រកមុំ $\theta$៖ $\cos\theta = \frac{a}{r} = -\frac{\sqrt{3}}{2}$ និង $\sin\theta = \frac{b}{r} = \frac{1}{2}$។
ដោយសារ $\cos\theta < 0$ និង $\sin\theta > 0$ នាំឱ្យមុំស្ថិតក្នុងកាដ្រង់ទី២៖ $\theta = \pi - \frac{\pi}{6} = \frac{5\pi}{6}$។
២០. ក្នុងប្លង់កុំផ្លិច សំណុំចំណុច $M(z)$ ដែលផ្ទៀងផ្ទាត់លក្ខខណ្ឌ $|z - i| = 2$ គឺជាអ្វី? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការទម្រង់ $|z - z_0| = R$ គឺជាសំណុំចំណុចរង្វង់ដែលមានផ្ចិតជាចំណុចរូបភាពនៃ $z_0$ និងកាំស្មើ $R$។
ទីនេះ $z_0 = i \implies$ កូអរដោនេផ្ចិតគឺ $(0, 1)$ ហើយកាំ $R = 2$។
២១. តើគេអាចជ្រើសរើសគណៈកម្មការមួយមានសមាជិក ៣នាក់ ពីចំណោមសិស្ស ១០នាក់ បានចំនួនប៉ុន្មានរបៀប? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ការជ្រើសរើសសមាជិកគណៈកម្មការ មិនគិតពីលំដាប់លំដោយទេ ដូចនេះយើងប្រើ បន្សំ (Combination)៖
$C(10, 3) = \frac{10!}{3!(10-3)!} = \frac{10 \times 9 \times 8}{3 \times 2 \times 1} = 120$ របៀប។

២៣. រកកូអរដោនេផ្ចិតនៃអេលីបដែលមានសមីការ៖ $\frac{(x-1)^2}{9} + \frac{(y+2)^2}{4} = 1$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការស្តង់ដានៃអេលីបគឺ៖ $\frac{(x-h)^2}{a^2} + \frac{(y-k)^2}{b^2} = 1$ ដែលមានផ្ចិត $I(h, k)$។
តាមសមីការ៖ $h = 1$ និង $k = -2$។ ដូចនេះ កូអរដោនេផ្ចិតគឺ $I(1, -2)$។
២៤. ក្នុងលំហកូអរដោនេ គណនាផលគុណវ៉ិចទ័រ $\vec{u} \times \vec{v}$ បើដឹងថា $\vec{u} = (1, 0, 1)$ និង $\vec{v} = (0, 1, 0)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេទែមីណង់សម្រាប់ផលគុណវ៉ិចទ័រ៖
$\vec{u} \times \vec{v} = \left( \begin{vmatrix} 0 & 1 \\ 1 & 0 \end{vmatrix}, \begin{vmatrix} 1 & 1 \\ 0 & 0 \end{vmatrix}, \begin{vmatrix} 1 & 0 \\ 0 & 1 \end{vmatrix} \right)$
$= (0\cdot0 - 1\cdot1, \, -(1\cdot0 - 1\cdot0), \, 1\cdot1 - 0\cdot0) = (-1, 0, 1)$។
២៥. គណនាចម្ងាយពីគល់កូអរដោនេ $O(0, 0, 0)$ ទៅប្លង់ $(P): x + y + z - 3 = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តចម្ងាយពីចំណុចទៅប្លង់៖ $d(O, P) = \frac{|a x_O + b y_O + c z_O + d|}{\sqrt{a^2 + b^2 + c^2}}$
$d(O, P) = \frac{|1(0) + 1(0) + 1(0) - 3|}{\sqrt{1^2 + 1^2 + 1^2}} = \frac{|-3|}{\sqrt{3}} = \frac{3}{\sqrt{3}} = \sqrt{3}$។
