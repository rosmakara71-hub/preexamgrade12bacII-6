ពិន្ទុទទួលបាន៖ 0 / ១២៥

👇 សូមពិនិត្យវិធីសាស្ត្រដោះស្រាយលម្អិតនៅខាងក្រោមសំនួរនីមួយៗ!

១. គណនាលីមីត៖ $\lim_{x \to 3} \frac{x^2 - 9}{x^2 - 3x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
រាងមិនកំណត់ $\frac{0}{0}$។ ដាក់ភាគយក និងភាគបែងជាកត្តា៖
$\lim_{x \to 3} \frac{(x-3)(x+3)}{x(x-3)} = \lim_{x \to 3} \frac{x+3}{x}$
$= \frac{3+3}{3} = \frac{6}{3} = 2$។
២. គណនាលីមីត៖ $\lim_{x \to 0} \frac{1 - \cos 2x}{x^2}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តត្រីកោណមាត្រ $1 - \cos 2x = 2\sin^2 x$៖
$\lim_{x \to 0} \frac{2\sin^2 x}{x^2} = 2 \times \lim_{x \to 0} \left(\frac{\sin x}{x}\right)^2 = 2 \times 1^2 = 2$។
៣. គណនាលីមីត៖ $\lim_{x \to \infty} \left(\frac{x+2}{x}\right)^x$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សរសេរកន្សោមឡើងវិញជាទម្រង់ $1^\infty$៖
$\lim_{x \to \infty} \left(1 + \frac{2}{x}\right)^x = \lim_{x \to \infty} \left[\left(1 + \frac{2}{x}\right)^{\frac{x}{2}}\right]^2 = e^2$។
៤. រកតម្លៃ $k$ ដើម្បីឱ្យអនុគមន៍ $f(x) = \begin{cases} \frac{\sin 4x}{2x} & \text{បើ } x \neq 0 \\ k & \text{បើ } x = 0 \end{cases}$ ជាប់ត្រង់ $x = 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ដើម្បីឱ្យអនុគមន៍ជាប់ត្រង់ $x=0$ លុះត្រាតែ $\lim_{x \to 0} f(x) = f(0) = k$។
គណនាលីមីត៖ $\lim_{x \to 0} \frac{\sin 4x}{2x} = \lim_{x \to 0} \left(\frac{\sin 4x}{4x} \times 2\right) = 1 \times 2 = 2$។
ដូចនេះ $k = 2$។
៥. រកដេរីវេនៃអនុគមន៍៖ $f(x) = \ln(\sin x)$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេបណ្តាក់ $(\ln u)' = \frac{u'}{u}$ ដែល $u = \sin x \implies u' = \cos x$។
$f'(x) = \frac{(\sin x)'}{\sin x} = \frac{\cos x}{\sin x} = \cot x$។
៦. រកដេរីវេនៃអនុគមន៍៖ $f(x) = x^2 e^{-x}$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ប្រើរូបមន្តដេរីវេផលគុណ $(uv)' = u'v + uv'$៖
$f'(x) = (x^2)'e^{-x} + x^2(e^{-x})' = 2xe^{-x} + x^2(-e^{-x})$
$= 2xe^{-x} - x^2e^{-x} = e^{-x}(2x - x^2)$។
៧. រកសមីការបន្ទាត់ប៉ះខ្សែគោង $y = x^2 - 3x + 2$ ត្រង់ចំណុចដែលមានអាប់ស៊ីស $x = 2$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការបន្ទាត់ប៉ះមានទម្រង់៖ $y = f'(2)(x - 2) + f(2)$
- គណនា $f(2) = 2^2 - 3(2) + 2 = 4 - 6 + 2 = 0$។
- រកដេរីវេ $f'(x) = 2x - 3 \implies f'(2) = 2(2) - 3 = 1$។
នាំឱ្យបាន៖ $y = 1(x - 2) + 0 \implies y = x - 2$។
៨. រកតម្លៃអប្បបរមារបស់អនុគមន៍ $f(x) = x + \frac{4}{x}$ ចំពោះ $x > 0$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
របៀបទី១៖ តាមវិសមភាព AM-GM (Cauchy) ចំពោះ $x > 0$៖
$x + \frac{4}{x} \ge 2\sqrt{x \times \frac{4}{x}} = 2\sqrt{4} = 4$។
របៀបទី២៖ រកដេរីវេ $f'(x) = 1 - \frac{4}{x^2} = 0 \implies x^2 = 4 \implies x = 2$ (ព្រោះ $x>0$)។ តម្លៃអប្បបរមាគឺ $f(2) = 2 + \frac{4}{2} = 4$។
៩. គណនាអាំងតេក្រាល៖ $\int 2x e^{x^2} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = x^2 \implies du = 2x \,dx$។
នាំឱ្យអាំងតេក្រាលទៅជា៖ $\int e^u \,du = e^u + C = e^{x^2} + C$។
១១. គណនាអាំងតេក្រាលកំណត់៖ $\int_{0}^{\frac{\pi}{2}} \sin x \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ព្រីមីទីវនៃ $\sin x$ គឺ $-\cos x$។
$\int_{0}^{\frac{\pi}{2}} \sin x \,dx = \left[-\cos x\right]_{0}^{\frac{\pi}{2}} = \left(-\cos\frac{\pi}{2}\right) - (-\cos 0) = 0 - (-1) = 1$។
១១. គណនាអាំងតេក្រាលកំណត់៖ $\int_{1}^{e} \frac{\ln x}{x} \,dx$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាង $u = \ln x \implies du = \frac{1}{x}dx$។
ប្តូរលីមីត៖ បើ $x=1 \implies u=0$; បើ $x=e \implies u=1$។
$\int_{0}^{1} u \,du = \left[\frac{u^2}{2}\right]_{0}^{1} = \frac{1}{2} - 0 = \frac{1}{2}$។
១២. គណនាផ្ទៃក្រឡាខណ្ឌដោយខ្សែគោង $y = x^2$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x = 0$ ដល់ $x = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ផ្ទៃក្រឡា៖ $S = \int_{0}^{3} x^2 \,dx = \left[\frac{x^3}{3}\right]_{0}^{3} = \frac{3^3}{3} - 0 = \frac{27}{3} = 9$ ឯកតាផ្ទៃ។
១៣. គណនាមាឌសូលីដបដិវត្តន៍បង្កើតឡើងដោយបង្វិលជុំវិញអ័ក្ស $Ox$ នៃផ្ទៃខណ្ឌដោយខ្សែគោង $y = \sqrt{x}$ អ័ក្សអាប់ស៊ីស និងបន្ទាត់ឈរ $x=0$ ដល់ $x=4$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
តាមរូបមន្តមាឌ៖ $V = \pi \int_{0}^{4} [f(x)]^2 \,dx = \pi \int_{0}^{4} (\sqrt{x})^2 \,dx = \pi \int_{0}^{4} x \,dx$។
$= \pi \left[\frac{x^2}{2}\right]_{0}^{4} = \pi \left(\frac{16}{2} - 0\right) = 8\pi$ ឯកតាមាឌ។
១៤. រកចម្លើយពិសេសនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y' - 2y = 0$ បើដឹងថា $y(0) = 3$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការ $y' - 2y = 0 \implies y' = 2y$ មានចម្លើយទូទៅ៖ $y = Ke^{2x}$។
ប្រើលក្ខខណ្ឌដើម $y(0) = 3 \implies 3 = Ke^{2(0)} \implies K = 3$។
ដូចនេះ ចម្លើយពិសេសគឺ $y = 3e^{2x}$។
១៥. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' + 9y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 + 9 = 0 \implies r^2 = -9 \implies r = \pm 3i$ (ឫសជាចំនួនកុំផ្លិចឆ្លាស់ $\alpha = 0, \beta = 3$)។
ដូចនេះ ចម្លើយទូទៅគឺ៖ $y = e^{0x}(A\cos 3x + B\sin 3x) = A\cos 3x + B\sin 3x$។
១៦. រកចម្លើយទូទៅនៃសមីការឌីផេរ៉ង់ស្យែល៖ $y'' - 2y' + y = 0$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការលក្ខណៈ៖ $r^2 - 2r + 1 = 0 \implies (r-1)^2 = 0 \implies r_1 = r_2 = 1$ (ឫសឌុប)។
ដូចនេះ ចម្លើយទូទៅគឺ៖ $y = (Ax + B)e^{1x} = (Ax + B)e^x$។
១៧. គេឱ្យចំនួនកុំផ្លិច $z = (1+2i)(2-i)$។ រកផ្នែកពិត $(\text{Re}(z))$ នៃចំនួនកុំផ្លិចនេះ។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ពង្រាត់កន្សោម $z$ ៖
$z = 1(2) - 1(i) + 2i(2) - 2i(i) = 2 - i + 4i - 2i^2$
ដោយសារ $i^2 = -1 \implies z = 2 + 3i - 2(-1) = 2 + 3i + 2 = 4 + 3i$។
ដូចនេះ ផ្នែកពិតគឺ $4$។
១៨. រកអាគុយម៉ង់ចម្បងនៃចំនួនកុំផ្លិច $z = -1 + i\sqrt{3}$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ម៉ូឌុល៖ $r = \sqrt{(-1)^2 + (\sqrt{3})^2} = \sqrt{1 + 3} = 2$។
- រកមុំ $\theta$៖ $\cos\theta = \frac{a}{r} = -\frac{1}{2}$ និង $\sin\theta = \frac{b}{r} = \frac{\sqrt{3}}{2}$។
ដោយសារ $\cos\theta < 0$ និង $\sin\theta > 0$ នាំឱ្យមុំស្ថិតក្នុងកាដ្រង់ទី២៖ $\theta = \pi - \frac{\pi}{3} = \frac{2\pi}{3}$។
១៩. ក្នុងប្លង់កុំផ្លិច សំណុំចំណុច $M(z)$ ដែលផ្ទៀងផ្ទាត់លក្ខខណ្ឌ $|z - i| = |z + i|$ គឺជាអ្វី? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
លក្ខខណ្ឌ $|z - i| = |z - (-i)|$ មានន័យថា ចម្ងាយពីចំណុច $M(z)$ ទៅចំណុច $A(i)$ ស្មើនឹងចម្ងាយពី $M(z)$ ទៅចំណុច $B(-i)$។
សំណុំចំណុច $M$ គឺជាបន្ទាត់មេដ្យាទ័រនៃអង្កត់ $AB$។ ដោយសារ $A$ និង $B$ ស្ថិតនៅលើអ័ក្សអរដោនេឆ្លុះគ្នាធៀបនឹងគល់ $O$ នោះបន្ទាត់មេដ្យាទ័ររបស់វាគឺ **អ័ក្សអាប់ស៊ីស ($y = 0$)**។
២០. ក្នុងថ្នាក់រៀនមួយមានសិស្ស ១០ នាក់។ តើគេអាចជ្រើសរើសគណៈកម្មការមួយក្រុមដែលមានសិស្ស ៣ នាក់បានប៉ុន្មានរបៀប? (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ការជ្រើសរើសមនុស្ស ៣ នាក់ចេញពី ១០ នាក់ដោយមិនគិតលំដាប់ គឺជាបន្ទំ (Combination)៖
$C(10, 3) = \frac{10!}{3!(10-3)!} = \frac{10 \times 9 \times 8}{3 \times 2 \times 1} = 120$ របៀប។
២១. ថង់មួយមានឃ្លីពណ៌ក្រហម ៤ គ្រាប់ និងឃ្លីពណ៌ខៀវ ៦ គ្រាប់។ គេចាប់យកឃ្លី ២ គ្រាប់ព្រមគ្នាដោយចៃដន្យ។ រកប្រូបាបដើម្បីបានឃ្លីពណ៌ក្រហមទាំង ២ គ្រាប់។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
- ចំនួនករណីអាច៖ $n(S) = C(10, 2) = \frac{10 \times 9}{2} = 45$
- ចំនួនករណីស្រប (ក្រហមទាំង២)៖ $n(A) = C(4, 2) = \frac{4 \times 3}{2} = 6$
- ប្រូបាប៖ $P(A) = \frac{n(A)}{n(S)} = \frac{6}{45} = \frac{2}{15}$។
២២. រកអុិចសង់ទ្រីស៊ីតេ $(e)$ នៃអេលីបដែលមានសមីការស្តង់ដា៖ $\frac{x^2}{25} + \frac{y^2}{16} = 1$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
ពីសមីការអេលីប៖ $a^2 = 25 \implies a = 5$ និង $b^2 = 16 \implies b = 4$។
គណនាចម្ងាយពីផ្ចិតទៅកំនុំ $c = \sqrt{a^2 - b^2} = \sqrt{25 - 16} = \sqrt{9} = 3$។
អុិចសង់ទ្រីស៊ីតេ៖ $e = \frac{c}{a} = \frac{3}{5}$។
២៣. រកសមីការអាស៊ីមតូតទាំងពីរនៃអ៊ីពែបូល៖ $\frac{x^2}{9} - \frac{y^2}{4} = 1$ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
អ៊ីពែបូលមានអ័ក្សទទឹងដេក (ព្រោះតួ $x^2$ វិជ្ជមាន) ដែលមាន $a^2 = 9 \implies a = 3$ និង $b^2 = 4 \implies b = 2$។
រូបមន្តសមីការអាស៊ីមតូតគឺ៖ $y = \pm \frac{b}{a}x$។
នាំឱ្យបាន៖ $y = \pm \frac{2}{3}x$។
២៤. ក្នុងលំហកូអរដោនេ រកសមីការប៉ារ៉ាម៉ែត្រនៃបន្ទាត់ដែលកាត់តាមចំណុច $A(1, 2, 3)$ និងមានវ៉ិចទ័រប្រាប់ទិស $\vec{u} = (2, -1, 4)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការប៉ារ៉ាម៉ែត្រនៃបន្ទាត់កាត់តាម $(x_0, y_0, z_0)$ និងមានវ៉ិចទ័រប្រាប់ទិស $(a, b, c)$ មានទម្រង់៖
$\begin{cases} x = x_0 + at \\ y = y_0 + bt \\ z = z_0 + ct \end{cases} (t \in \mathbb{R})$
ជំនួសតម្លៃចូល៖ $\begin{cases} x = 1 + 2t \\ y = 2 - t \\ z = 3 + 4t \end{cases}$។
២៥. រកសមីការប្លង់ដែលកាត់តាមចំណុច $P(2, 1, -1)$ និងកែងនឹងវ៉ិចទ័រណរម៉ាល់ $\vec{n} = (1, 3, -2)$។ (៥ ពិន្ទុ)
💡 ដំណោះស្រាយលម្អិត៖
សមីការប្លង់មានទម្រង់៖ $a(x - x_0) + b(y - y_0) + c(z - z_0) = 0$
ជំនួស $a=1, b=3, c=-2$ និង $x_0=2, y_0=1, z_0=-1$ ៖
$1(x - 2) + 3(y - 1) - 2(z - (-1)) = 0$
$x - 2 + 3y - 3 - 2z - 2 = 0 \implies x + 3y - 2z - 7 = 0$។
